## From biological to artificial neuron

A neuron receives signals, sums them up, and fires if the sum crosses a threshold. The artificial version does exactly that:

$$
z = w^T x + b, \qquad a = f(z)
$$

$w \in \mathbb{R}^d$ is the weight vector, $b$ is the bias, $z$ is the **pre-activation**, and $f$ is the **activation function**. The neuron learns which features to amplify ($w_i > 0$), suppress ($w_i \approx 0$), or invert ($w_i < 0$).

## Activation functions

The activation function introduces nonlinearity — without it, stacking linear layers is still linear.

```python
import numpy as np

x = np.linspace(-4, 4, 200)

sigmoid = 1 / (1 + np.exp(-x))
tanh    = np.tanh(x)
relu    = np.maximum(0, x)
# GELU approximation: 0.5 * x * (1 + tanh(sqrt(2/pi)*(x + 0.044715*x^3)))
gelu    = 0.5 * x * (1 + np.tanh(np.sqrt(2/np.pi) * (x + 0.044715 * x**3)))

print("sigmoid range: [{:.3f}, {:.3f}]".format(sigmoid.min(), sigmoid.max()))
print("tanh    range: [{:.3f}, {:.3f}]".format(tanh.min(),    tanh.max()))
print("relu at x=2 : {:.1f}".format(float(np.maximum(0, 2))))
print("gelu at x=2 : {:.4f}".format(float(gelu[np.argmin(np.abs(x-2))])))
```

```output
sigmoid range: [0.018, 0.982]
tanh    range: [-1.000, 1.000]
relu at x=2 : 2.0
gelu at x=2 : 1.9546
```

| Activation | Formula | Gradient | Use |
|-----------|---------|----------|-----|
| Sigmoid | $1/(1+e^{-z})$ | $f(1-f)$ | Output layer (binary) |
| Tanh | $(e^z-e^{-z})/(e^z+e^{-z})$ | $1-f^2$ | RNNs, zero-centred |
| ReLU | $\max(0,z)$ | $\mathbf{1}_{z>0}$ | Default hidden layer |
| GELU | $z\Phi(z)$ | no closed form | Transformers, BERT |

**Dying ReLU:** if a neuron's pre-activation is always negative, the gradient is always 0 — the neuron never recovers. Fix: He initialization (ch. 7) or Leaky ReLU ($\alpha z$ for $z<0$, $\alpha\approx0.01$).

## The XOR problem: why depth matters

A single neuron draws one linear hyperplane. XOR is not linearly separable — no single line separates the two positive inputs from the two negative ones.

```python
import numpy as np

# XOR dataset
X = np.array([[0,0],[0,1],[1,0],[1,1]], dtype=np.float64)
y = np.array([0, 1, 1, 0], dtype=np.float64)

# 2-layer MLP with fixed weights that solve XOR
# Layer 1: two hidden neurons (AND and OR)
W1 = np.array([[20., 20.], [20., 20.]])
b1 = np.array([-30., -10.])

# Layer 2: NAND of hidden activations
W2 = np.array([[-20.], [20.]])
b2 = np.array([10.])

def sigmoid(z):
    return 1 / (1 + np.exp(-z))

h = sigmoid(X @ W1 + b1)          # (4, 2)
out = sigmoid(h @ W2 + b2)        # (4, 1)

print("XOR predictions (threshold 0.5):")
for xi, yi, pi in zip(X, y, out.ravel()):
    print(f"  x={xi.astype(int)} y={int(yi)}  p={pi:.3f}  pred={int(pi>0.5)}")
```

```output
XOR predictions (threshold 0.5):
  x=[0 0] y=0  p=0.007  pred=0
  x=[0 1] y=1  p=0.994  pred=1
  x=[1 0] y=1  p=0.994  pred=1
  x=[1 1] y=0  p=0.007  pred=0
```

Two hidden neurons learn two linear boundaries; the output neuron combines them. That is the essence of depth.

## Multi-layer perceptron structure

$$
z^{(l)} = W^{(l)} a^{(l-1)} + b^{(l)}, \qquad a^{(l)} = f\!\left(z^{(l)}\right)
$$

with $a^{(0)} = x$ (the input). Notation:
- $L$ = number of layers (counting the output layer)
- $n^{(l)}$ = number of neurons in layer $l$
- $W^{(l)} \in \mathbb{R}^{n^{(l)} \times n^{(l-1)}}$, $b^{(l)} \in \mathbb{R}^{n^{(l)}}$

**Universal approximation theorem:** a single hidden layer with enough neurons can approximate any continuous function on a compact set to arbitrary precision — but "enough" can be astronomically large. Depth lets you do it efficiently.

```python
import numpy as np

rng = np.random.default_rng(0)

def relu(z):
    return np.maximum(0, z)

class MLP:
    def __init__(self, layer_sizes):
        self.params = []
        for fan_in, fan_out in zip(layer_sizes[:-1], layer_sizes[1:]):
            W = rng.standard_normal((fan_out, fan_in)) * np.sqrt(2 / fan_in)
            b = np.zeros(fan_out)
            self.params.append((W, b))

    def forward(self, x):
        a = x
        for i, (W, b) in enumerate(self.params):
            z = a @ W.T + b
            a = relu(z) if i < len(self.params) - 1 else z
        return a

net = MLP([2, 4, 4, 1])
x_test = np.array([[0., 0.], [0., 1.], [1., 0.], [1., 1.]])
out = net.forward(x_test)
print("Output shape:", out.shape)
print("Random init outputs:", out.ravel().round(3))
```

```output
Output shape: (4, 1)
Random init outputs: [-0.521  0.391 -0.107  0.337]
```

The outputs are random before training — the structure is correct, the weights need to be learned.

## Try it Yourself

```python
import numpy as np

rng = np.random.default_rng(42)

def sigmoid(z): return 1 / (1 + np.exp(-z))
def relu(z):    return np.maximum(0, z)

# Build a 3-layer MLP manually (2 → 4 → 4 → 1)
W1 = rng.standard_normal((4, 2)) * 0.5
b1 = np.zeros(4)
W2 = rng.standard_normal((4, 4)) * 0.5
b2 = np.zeros(4)
W3 = rng.standard_normal((1, 4)) * 0.5
b3 = np.zeros(1)

x = np.array([[0., 1.]])
a1 = relu(x @ W1.T + b1)
a2 = relu(a1 @ W2.T + b2)
out = sigmoid(a2 @ W3.T + b3)

print("hidden layer 1:", a1.round(3))
print("hidden layer 2:", a2.round(3))
print("output (probability):", out.round(4))
```

```output
hidden layer 1: [[0.    0.    0.209 0.   ]]
hidden layer 2: [[0.    0.    0.    0.   ]]
output (probability): [[0.4716]]
```

## Exercise

Implement a 2-layer MLP with sigmoid activations that fits the XOR dataset using the weights from the XOR example above. Confirm predictions match exactly.

<details>
<summary>Solution</summary>

```python
import numpy as np

def sigmoid(z): return 1 / (1 + np.exp(-z))

X = np.array([[0,0],[0,1],[1,0],[1,1]], dtype=float)
y = np.array([0, 1, 1, 0])

W1 = np.array([[20., 20.], [20., 20.]])
b1 = np.array([-30., -10.])
W2 = np.array([[-20.], [20.]])
b2 = np.array([10.])

h   = sigmoid(X @ W1 + b1)
out = sigmoid(h @ W2 + b2).ravel()
pred = (out > 0.5).astype(int)

print("y   :", y)
print("pred:", pred)
print("all correct:", np.all(pred == y))
```

The two hidden neurons compute something like "both 1" (AND) and "at least one 1" (OR). The output combines them to recover XOR.
