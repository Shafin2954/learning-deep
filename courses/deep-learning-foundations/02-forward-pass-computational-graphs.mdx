## Computational graphs

Every computation in a neural network can be drawn as a **directed acyclic graph (DAG)**: leaf nodes are inputs and parameters, internal nodes are operations, edges carry values. This representation makes automatic differentiation mechanical.

For the scalar loss $L = (w_1 x_1 + w_2 x_2 - y)^2$, the graph is:

```
x1, w1 → mul → a
x2, w2 → mul → b
a, b   → add → s
s, y   → sub → e
e      → sq  → L
```

The forward pass evaluates this graph left to right, storing every intermediate value. Storage is the price you pay to make backward efficient.

## A scalar Value class

Build the graph node-by-node:

```python
class Value:
    def __init__(self, data, _children=(), _op=''):
        self.data = float(data)
        self.grad = 0.0
        self._prev = set(_children)
        self._op   = _op

    def __add__(self, other):
        other = other if isinstance(other, Value) else Value(other)
        out = Value(self.data + other.data, (self, other), '+')
        return out

    def __mul__(self, other):
        other = other if isinstance(other, Value) else Value(other)
        out = Value(self.data * other.data, (self, other), '*')
        return out

    def __pow__(self, exponent):
        assert isinstance(exponent, (int, float))
        out = Value(self.data ** exponent, (self,), f'**{exponent}')
        return out

    def __repr__(self):
        return f"Value(data={self.data:.4f}, grad={self.grad:.4f})"

# Build: L = (w1*x1 + w2*x2 - y)^2
x1, x2 = Value(2.0), Value(3.0)
w1, w2 = Value(0.5), Value(-0.3)
y_true  = Value(1.0)

s = w1 * x1 + w2 * x2
e = s + Value(-1.0) * y_true   # subtract y
L = e ** 2

print("x1:", x1)
print("x2:", x2)
print("s (pre-output):", s)
print("e (error):",      e)
print("L (loss):",       L)
```

```output
x1: Value(data=2.0000, grad=0.0000)
x2: Value(data=3.0000, grad=0.0000)
s (pre-output): Value(data=0.1000, grad=0.0000)
e (error): Value(data=-0.9000, grad=0.0000)
L (loss): Value(data=0.8100, grad=0.0000)
```

Every node stores its data and a `grad` slot (initially 0) for the gradient that backprop will fill in.

## Matrix form of the forward pass

In practice you compute with batches. For a layer with $n^{(l)}$ neurons processing a batch of $N$ samples:

$$
Z^{(l)} = A^{(l-1)} W^{(l)T} + \mathbf{1} b^{(l)T}, \qquad A^{(l)} = f(Z^{(l)})
$$

where $A^{(l-1)} \in \mathbb{R}^{N \times n^{(l-1)}}$ and $Z^{(l)} \in \mathbb{R}^{N \times n^{(l)}}$.

```python
import numpy as np

rng = np.random.default_rng(1)

N, d_in, d_h, d_out = 32, 4, 8, 1

# Layer 1: (d_in → d_h)
W1 = rng.standard_normal((d_h, d_in)) * np.sqrt(2 / d_in)
b1 = np.zeros(d_h)

# Layer 2: (d_h → d_out)
W2 = rng.standard_normal((d_out, d_h)) * np.sqrt(2 / d_h)
b2 = np.zeros(d_out)

X = rng.standard_normal((N, d_in))

# Forward pass — store intermediates
Z1 = X  @ W1.T + b1           # (N, d_h)
A1 = np.maximum(0, Z1)        # ReLU
Z2 = A1 @ W2.T + b2           # (N, d_out)
# output is Z2 (linear, regression)

print("X  shape:", X.shape)
print("Z1 shape:", Z1.shape, "  A1 shape:", A1.shape)
print("Z2 shape:", Z2.shape)
print("Z2 stats: mean={:.3f}  std={:.3f}".format(Z2.mean(), Z2.std()))
```

```output
X  shape: (32, 4)
Z1 shape: (32, 8)   A1 shape: (32, 8)
Z2 shape: (32, 1)
Z2 stats: mean=0.033  std=0.568
```

You must store `X`, `Z1`, `A1` during the forward pass — backpropagation needs them to compute gradients.

### Why you store intermediates

The gradient of the loss with respect to $W^{(1)}$ requires $A^{(0)} = X$ (the input to layer 1). The gradient with respect to $W^{(2)}$ requires $A^{(1)}$. Each layer needs its *input* from the forward pass. This is the memory cost of backprop: $O(N \times \sum n^{(l)})$.

## Try it Yourself

```python
import numpy as np

rng = np.random.default_rng(7)

# 3-layer forward pass: [3 → 5 → 5 → 2]
layer_sizes = [3, 5, 5, 2]
Ws = [rng.standard_normal((layer_sizes[i+1], layer_sizes[i])) * 0.5
      for i in range(len(layer_sizes)-1)]
bs = [np.zeros(layer_sizes[i+1]) for i in range(len(layer_sizes)-1)]

X_batch = rng.standard_normal((8, 3))

cache = {'A0': X_batch}
A = X_batch
for l, (W, b) in enumerate(zip(Ws, bs)):
    Z = A @ W.T + b
    A = np.maximum(0, Z) if l < len(Ws)-1 else Z
    cache[f'Z{l+1}'] = Z
    cache[f'A{l+1}'] = A

print("Stored cache keys:", list(cache.keys()))
print("Final output A3:", A.round(3))
```

```output
Stored cache keys: ['A0', 'Z1', 'A1', 'Z2', 'A2', 'Z3', 'A3']
Final output A3: [[ 0.037 -0.127]
 [ 0.283  0.078]
 [ 0.    -0.   ]
 [-0.157 -0.085]
 [ 0.14  -0.104]
 [-0.007  0.104]
 [ 0.054  0.059]
 [-0.187 -0.056]]
```

## Exercise

Implement a forward pass for a 2-layer MLP $(4 \to 8 \to 1)$ with ReLU on the hidden layer and no activation on the output. Return both the prediction and a cache dict with all intermediates. Compute MSE loss between predictions and a target vector.

<details>
<summary>Solution</summary>

```python
import numpy as np

rng = np.random.default_rng(0)

W1 = rng.standard_normal((8, 4)) * np.sqrt(2/4)
b1 = np.zeros(8)
W2 = rng.standard_normal((1, 8)) * np.sqrt(2/8)
b2 = np.zeros(1)

X = rng.standard_normal((16, 4))
y = rng.standard_normal((16, 1))

Z1 = X  @ W1.T + b1
A1 = np.maximum(0, Z1)
Z2 = A1 @ W2.T + b2     # (16, 1) predictions

cache = {'X': X, 'Z1': Z1, 'A1': A1, 'Z2': Z2}
loss = np.mean((Z2 - y) ** 2)

print("Prediction shape:", Z2.shape)
print("MSE loss:", round(float(loss), 4))
print("Cache keys:", list(cache.keys()))
```

All intermediate values are stored; `Z2` is the prediction, `loss` is MSE.
</details>
