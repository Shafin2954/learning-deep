## Joint distributions

Two random variables $X$ and $Y$ are described jointly by their **joint PDF** $f(x,y)$ (continuous) or **joint PMF** $P(X=x, Y=y)$ (discrete). The joint distribution encodes everything about the pair, including how they depend on each other.

The joint PDF must satisfy $f(x,y) \ge 0$ and $\int\!\int f(x,y)\,dx\,dy = 1$.

```python
import numpy as np
from scipy import stats

# Bivariate Normal: joint distribution of two correlated Normals
rho = 0.7  # correlation
mean = [0, 0]
cov  = [[1, rho], [rho, 1]]

dist = stats.multivariate_normal(mean=mean, cov=cov)

# Evaluate joint PDF at a grid point
x, y = 0.5, 0.5
print(f"f(0.5, 0.5) = {dist.pdf([x, y]):.6f}")

# Sample and verify
rng = np.random.default_rng(0)
samples = rng.multivariate_normal(mean, cov, size=10_000)
print(f"Sample corr: {np.corrcoef(samples[:,0], samples[:,1])[0,1]:.4f}  (true: {rho})")
```

```output
f(0.5, 0.5) = 0.112871
Sample corr: 0.7027  (true: 0.7)
```

## Marginal distributions

The **marginal distribution** of $X$ is obtained by integrating (or summing) out $Y$:

$$
f_X(x) = \int_{-\infty}^{\infty} f(x,y)\,dy
$$

For the bivariate Normal with $\rho=0.7$, the marginal of $X$ is $N(0,1)$ — the joint correlation "disappears" when you look at $X$ alone.

```python
import numpy as np
from scipy import stats

rho = 0.7
mean = [0, 0]
cov  = [[1, rho], [rho, 1]]

rng = np.random.default_rng(1)
samples = rng.multivariate_normal(mean, cov, size=20_000)
x_samples = samples[:, 0]
y_samples = samples[:, 1]

# Marginal of X should be N(0,1)
print(f"X marginal: mean={x_samples.mean():.4f}  std={x_samples.std():.4f}")
print(f"Y marginal: mean={y_samples.mean():.4f}  std={y_samples.std():.4f}")
print(f"Theoretical (both): mean=0  std=1")
```

```output
X marginal: mean=-0.0035  std=0.9981
Y marginal: mean=0.0071  std=1.0018
Theoretical (both): mean=0  std=1
```

## Conditional distributions

The conditional distribution of $Y$ given $X=x$ is:

$$
f(y \mid x) = \frac{f(x,y)}{f_X(x)}
$$

For bivariate Normal, the conditional $Y \mid X=x$ is still Normal:

$$
Y \mid X = x \;\sim\; N\!\left(\mu_Y + \rho\frac{\sigma_Y}{\sigma_X}(x-\mu_X),\;\sigma_Y^2(1-\rho^2)\right)
$$

```python
import numpy as np

rho = 0.7
# Conditional Y | X=1.5  (both marginals are N(0,1))
x_val = 1.5
cond_mean = rho * x_val          # = 1.05
cond_std  = np.sqrt(1 - rho**2)  # ≈ 0.714

print(f"Y | X=1.5 ~ N({cond_mean:.3f}, {cond_std:.3f}²)")

# Verify by simulation: keep only samples where X ≈ 1.5
rng = np.random.default_rng(2)
samples = rng.multivariate_normal([0,0], [[1,rho],[rho,1]], size=500_000)
mask = np.abs(samples[:,0] - x_val) < 0.05
cond_samples = samples[mask, 1]
print(f"Empirical: mean={cond_samples.mean():.3f}  std={cond_samples.std():.3f}")
```

```output
Y | X=1.5 ~ N(1.050, 0.714²)
Empirical: mean=1.048  std=0.714
```

## Covariance and correlation

**Covariance** measures the direction and magnitude of linear dependence:

$$
\text{Cov}(X,Y) = E[XY] - E[X]E[Y]
$$

**Pearson correlation** normalizes to $[-1, 1]$:

$$
\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}
$$

```python
import numpy as np

rng = np.random.default_rng(3)
n = 5_000

# Create correlated variables
x = rng.standard_normal(n)
y = 0.7 * x + np.sqrt(1 - 0.49) * rng.standard_normal(n)

cov_xy = np.cov(x, y)          # 2×2 covariance matrix
print("Covariance matrix:")
print(cov_xy.round(4))

rho_hat = cov_xy[0,1] / np.sqrt(cov_xy[0,0] * cov_xy[1,1])
print(f"\nEstimated correlation: {rho_hat:.4f}  (true: 0.7000)")
```

```output
Covariance matrix:
[[1.0077 0.6997]
 [0.6997 0.9940]]

Estimated correlation: 0.6981  (true: 0.7000)
```

### Independence vs. uncorrelated

**Independence** implies $\text{Cov}(X,Y)=0$ (uncorrelated), but the converse is false. You can have $\text{Cov}=0$ while $X$ and $Y$ are strongly dependent.

```python
import numpy as np

rng = np.random.default_rng(4)
n = 10_000
x = rng.standard_normal(n)
y = x**2  # clearly depends on x, but...

cov = np.cov(x, y)[0, 1]
corr = np.corrcoef(x, y)[0, 1]

print(f"Cov(X, X²) = {cov:.4f}")
print(f"Corr(X, X²) = {corr:.4f}")
print("Uncorrelated but NOT independent (Y = X² ≠ constant * X)")
```

```output
Cov(X, X²) = -0.0016
Corr(X, X²) = -0.0006
Uncorrelated but NOT independent (Y = X² ≠ constant * X)
```

## Try it Yourself

```python
import numpy as np

rng = np.random.default_rng(99)
n = 8_000

# Simulate: X ~ N(2, 1), Y = 0.5*X + eps, eps ~ N(0, 0.5)
X = rng.normal(2, 1, n)
eps = rng.normal(0, np.sqrt(0.5), n)
Y = 0.5 * X + eps

print(f"E[X]   = {X.mean():.4f}  (true: 2.0)")
print(f"E[Y]   = {Y.mean():.4f}  (true: {0.5*2:.4f})")
print(f"Var[X] = {X.var():.4f}  (true: 1.0)")
print(f"Var[Y] = {Y.var():.4f}  (true: {0.5**2*1 + 0.5:.4f})")
print(f"Cov[X,Y] = {np.cov(X,Y)[0,1]:.4f}  (true: {0.5*1.0:.4f})")
print(f"Corr[X,Y] = {np.corrcoef(X,Y)[0,1]:.4f}")
```

```output
E[X]   = 2.0038  (true: 2.0)
E[Y]   = 1.0054  (true: 1.0000)
Var[X] = 0.9986  (true: 1.0)
Var[Y] = 0.7487  (true: 0.7500)
Cov[X,Y] = 0.5021  (true: 0.5000)
Corr[X,Y] = 0.5792
```

## Exercise

Two random variables $X\sim N(0,1)$ and $W\sim N(0,1)$ are independent. Define $Y = aX + bW$. (1) Express $\text{Cov}(X,Y)$ in terms of $a$ and $b$. (2) What value of $a/b$ gives $\rho_{XY}=0.6$? (3) Verify with a simulation of 20,000 samples.

<details>
<summary>Solution</summary>

```python
import numpy as np

# (1) Cov(X, Y) = Cov(X, aX + bW) = a*Var(X) + b*Cov(X,W) = a  (since Var(X)=1, Cov(X,W)=0)
# Var(Y) = a^2 + b^2 (since X,W independent, unit variance)
# Corr(X,Y) = a / sqrt(a^2 + b^2) = 0.6
# So a^2 / (a^2 + b^2) = 0.36  =>  b^2 = a^2 * (1/0.36 - 1) = a^2 * (64/36)
# b/a = 8/6 = 4/3  =>  a/b = 3/4 = 0.75

a, b = 0.6, 0.8  # satisfies a^2+b^2=1 and a/(sqrt(a^2+b^2))=0.6

rng = np.random.default_rng(10)
n = 20_000
X = rng.standard_normal(n)
W = rng.standard_normal(n)
Y = a * X + b * W

print(f"a={a}, b={b}")
print(f"Theoretical Corr(X,Y) = a / sqrt(a²+b²) = {a/np.sqrt(a**2+b**2):.4f}")
print(f"Empirical   Corr(X,Y) = {np.corrcoef(X,Y)[0,1]:.4f}")
```

$\text{Cov}(X,Y) = a$. For $\rho=0.6$, set $a=0.6, b=0.8$ (so $a^2+b^2=1$). The correlation equals $a/\sqrt{a^2+b^2}$.
</details>
