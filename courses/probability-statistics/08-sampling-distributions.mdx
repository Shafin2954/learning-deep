## The sampling distribution of the mean

Draw a sample of size $n$ from a population with mean $\mu$ and standard deviation $\sigma$. The sample mean $\bar{X}$ is itself a random variable — each new sample gives a different value.

Its distribution across repeated samples is the **sampling distribution** of $\bar{X}$:

$$
E[\bar{X}] = \mu, \qquad \text{Var}[\bar{X}] = \frac{\sigma^2}{n}, \qquad \text{SE}(\bar{X}) = \frac{\sigma}{\sqrt{n}}
$$

The standard error (SE) is the standard deviation of the sampling distribution. It measures how much $\bar{X}$ varies from sample to sample, not the spread of the underlying population.

```python
import numpy as np

rng = np.random.default_rng(0)
mu, sigma = 10.0, 4.0
n = 25
N_sims = 50_000

# Simulate sampling distribution of X_bar
xbars = rng.normal(mu, sigma, (N_sims, n)).mean(axis=1)

print(f"Population: mu={mu}, sigma={sigma}")
print(f"Sample size n={n}")
print(f"Theoretical SE = sigma/sqrt(n) = {sigma/np.sqrt(n):.4f}")
print(f"Empirical SE (std of x_bar)   = {xbars.std():.4f}")
print(f"Empirical mean of x_bar       = {xbars.mean():.4f}  (true: {mu})")
```

```output
Population: mu=10.0, sigma=4.0
Sample size n=25
Theoretical SE = sigma/sqrt(n) = 0.8000
Empirical SE (std of x_bar)   = 0.7989
Empirical mean of x_bar       = 9.9994  (true: 10.0)
```

## The t-distribution

When $\sigma$ is **unknown**, you estimate it with $S=\sqrt{\frac{1}{n-1}\sum(X_i-\bar{X})^2}$. The standardized mean now follows a **t-distribution** with $n-1$ degrees of freedom:

$$
T = \frac{\bar{X} - \mu}{S/\sqrt{n}} \;\sim\; t_{n-1}
$$

The t-distribution has heavier tails than the Normal — reflecting the added uncertainty from estimating $\sigma$. As $n\to\infty$ (equivalently df $\to\infty$), $t_{n-1}\to N(0,1)$.

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(1)
mu = 5.0
n  = 5        # small sample → t_4 distribution
N_sims = 50_000

t_stats = np.zeros(N_sims)
for i in range(N_sims):
    x = rng.normal(mu, 3.0, n)
    t_stats[i] = (x.mean() - mu) / (x.std(ddof=1) / np.sqrt(n))

# Compare percentiles to t(df=4) and N(0,1)
percs = [2.5, 5.0, 95.0, 97.5]
t4  = stats.t(df=n-1)
z   = stats.norm()

print(f"Percentile  Empirical     t({n-1})     N(0,1)")
print("-" * 50)
for p in percs:
    emp = np.percentile(t_stats, p)
    print(f"  {p:5.1f}%     {emp:7.4f}    {t4.ppf(p/100):7.4f}    {z.ppf(p/100):7.4f}")
```

```output
Percentile  Empirical     t(4)     N(0,1)
--------------------------------------------------
    2.5%     -2.7786    -2.7764    -1.9600
    5.0%     -2.1288    -2.1318    -1.6449
   95.0%      2.1341     2.1318     1.6449
   97.5%      2.7745     2.7764     1.9600
```

The t-distribution has wider tails: the 97.5th percentile is 2.78 (vs 1.96 for Normal).

## The chi-squared distribution

If $Z_1,\ldots,Z_k\overset{iid}{\sim}N(0,1)$, then $\chi^2_k = \sum_{i=1}^k Z_i^2 \sim \chi^2(k)$.

It arises naturally from the sample variance: $(n-1)S^2/\sigma^2 \sim \chi^2_{n-1}$.

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(2)
sigma = 2.0
n, N_sims = 10, 50_000

# Simulated chi-squared statistic from sample variances
chi2_stats = np.zeros(N_sims)
for i in range(N_sims):
    x = rng.normal(0, sigma, n)
    s2 = x.var(ddof=1)
    chi2_stats[i] = (n-1) * s2 / sigma**2

# Should follow chi2(df=n-1=9)
chi2_dist = stats.chi2(df=n-1)
print(f"(n-1)*S^2/sigma^2 ~ chi2({n-1})")
print(f"  Empirical mean: {chi2_stats.mean():.4f}  (true: {n-1})")
print(f"  Empirical var:  {chi2_stats.var():.4f}  (true: {2*(n-1)})")

ks_stat, p_val = stats.kstest(chi2_stats, chi2_dist.cdf)
print(f"  KS p-value: {p_val:.4f}  (consistent with chi2({n-1})? {p_val > 0.05})")
```

```output
(n-1)*S^2/sigma^2 ~ chi2(9)
  Empirical mean: 8.9971  (true: 9)
  Empirical var:  18.0073  (true: 18)
  KS p-value: 0.8121  (consistent with chi2(9)? True)
```

## The F-distribution

The ratio of two independent scaled chi-squared RVs is an F-distribution:

$$
F = \frac{\chi^2_{d_1}/d_1}{\chi^2_{d_2}/d_2} \;\sim\; F(d_1, d_2)
$$

It arises in ANOVA (comparing variances across groups) and regression (overall model significance).

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(3)
d1, d2 = 5, 10
N_sims = 50_000

f_stats = np.zeros(N_sims)
for i in range(N_sims):
    chi1 = rng.chisquare(d1)
    chi2 = rng.chisquare(d2)
    f_stats[i] = (chi1/d1) / (chi2/d2)

f_dist = stats.f(d1, d2)
print(f"F({d1},{d2}) distribution:")
print(f"  Empirical mean: {f_stats.mean():.4f}  (true: {d2/(d2-2):.4f})")

ks_stat, p_val = stats.kstest(f_stats, f_dist.cdf)
print(f"  KS p-value: {p_val:.4f}  (fits F({d1},{d2})? {p_val > 0.05})")
```

```output
F(5,10) distribution:
  Empirical mean: 1.2499  (true: 1.2500)
  KS p-value: 0.5217  (fits F(5,10)? True)
```

## Try it Yourself

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(42)
n = 8       # very small sample
N_sims = 30_000

# t-distribution: compare df=7, df=30, df=∞ (Normal) tail thickness
configs = [8, 30, 1000]
print(f"{'df':<8}  {'P(|T|>2.5)':<15}  {'t(0.975)':<12}  {'z(0.975)'}")
for n in configs:
    t_samps = np.array([
        (rng.normal(0,1,n).mean()) / (rng.normal(0,1,n).std(ddof=1)/np.sqrt(n))
        for _ in range(N_sims)
    ])
    tail_prob = (np.abs(t_samps) > 2.5).mean()
    crit_val  = stats.t(df=n-1).ppf(0.975)
    print(f"{n-1:<8}  {tail_prob:.5f}        {crit_val:.4f}        {stats.norm.ppf(0.975):.4f}")
```

```output
df        P(|T|>2.5)       t(0.975)      z(0.975)
7         0.04040          2.3646        1.9600
29        0.01883          2.0452        1.9600
999       0.01262          1.9623        1.9600
```

## Exercise

You have two groups: Group A with $n_A=15$ observations and Group B with $n_B=20$. Both sampled from Normal populations with the same $\sigma=5$. Simulate 20,000 values of $F=S_A^2/S_B^2$. (1) What is its theoretical distribution? (2) What fraction of values exceed 3.0?

<details>
<summary>Solution</summary>

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(10)
sigma = 5.0
nA, nB = 15, 20
N_sims = 20_000

f_vals = np.zeros(N_sims)
for i in range(N_sims):
    sa2 = rng.normal(0, sigma, nA).var(ddof=1)
    sb2 = rng.normal(0, sigma, nB).var(ddof=1)
    f_vals[i] = sa2 / sb2

# F ratio: (S_A^2/sigma^2) / (S_B^2/sigma^2) = S_A^2/S_B^2 ~ F(nA-1, nB-1)
f_dist = stats.f(df1=nA-1, df2=nB-1)
print(f"S_A^2 / S_B^2 ~ F({nA-1}, {nB-1})")
print(f"Fraction > 3.0:  empirical={( f_vals > 3.0).mean():.4f}  "
      f"theoretical={1-f_dist.cdf(3.0):.4f}")
```

The ratio $S_A^2/S_B^2 \sim F(n_A-1, n_B-1) = F(14, 19)$. Under equal variance the F ratio centers near 1; values far from 1 suggest the variances differ.
</details>
