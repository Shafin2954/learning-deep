## The moment generating function

The **moment generating function (MGF)** of a random variable $X$ is:

$$
M_X(t) = E\!\left[e^{tX}\right] = \int_{-\infty}^{\infty} e^{tx} f(x)\,dx
$$

when this expectation exists in a neighborhood of $t=0$. The key property: derivatives at $t=0$ give moments.

$$
E[X^k] = M_X^{(k)}(0) = \left.\frac{d^k}{dt^k} M_X(t)\right|_{t=0}
$$

This works because $e^{tX} = \sum_{k=0}^\infty \frac{(tX)^k}{k!}$, so $M_X(t) = \sum_{k=0}^\infty \frac{E[X^k]}{k!} t^k$.

### MGF of the Normal distribution

For $X\sim N(\mu,\sigma^2)$:

$$
M_X(t) = \exp\!\left(\mu t + \frac{\sigma^2 t^2}{2}\right)
$$

```python
import numpy as np
from scipy import stats

mu, sigma = 3.0, 2.0

# Analytic MGF and its derivatives at t=0
# M(t) = exp(mu*t + sigma^2*t^2/2)
# M'(t) = (mu + sigma^2*t) * M(t)  => M'(0) = mu
# M''(t) involves more terms; at t=0: M''(0) = mu^2 + sigma^2

# Numerically verify via Monte Carlo
rng = np.random.default_rng(0)
samples = rng.normal(mu, sigma, 1_000_000)

# E[e^{tX}] at t=0.1 — compare to analytic
t = 0.1
mc_mgf   = np.mean(np.exp(t * samples))
analytic = np.exp(mu*t + 0.5*sigma**2*t**2)
print(f"M({t}) MC={mc_mgf:.6f}  analytic={analytic:.6f}")

# Moments from analytic MGF
print(f"\nE[X]   = mu               = {mu:.4f}")
print(f"E[X²]  = mu²+sigma²        = {mu**2 + sigma**2:.4f}")
print(f"E[X³]  = mu³+3*mu*sigma²   = {mu**3 + 3*mu*sigma**2:.4f}")

# Compare to sample moments
print(f"\nSample E[X]  = {samples.mean():.4f}")
print(f"Sample E[X²] = {np.mean(samples**2):.4f}")
print(f"Sample E[X³] = {np.mean(samples**3):.4f}")
```

```output
M(0.1) MC=1.371856  analytic=1.371863

E[X]   = mu               = 3.0000
E[X²]  = mu²+sigma²        = 13.0000
E[X³]  = mu³+3*mu*sigma²   = 63.0000

Sample E[X]  = 3.0014
Sample E[X²] = 13.0093
Sample E[X³] = 63.2051
```

### MGF of common distributions

| Distribution | $M_X(t)$ |
|---|---|
| $\text{Bernoulli}(p)$ | $1-p+pe^t$ |
| $\text{Poisson}(\lambda)$ | $\exp(\lambda(e^t-1))$ |
| $\text{Exponential}(\lambda)$ | $\lambda/(\lambda-t)$, $t<\lambda$ |
| $N(\mu,\sigma^2)$ | $\exp(\mu t + \sigma^2 t^2/2)$ |

The MGF **uniquely characterizes** the distribution: if $M_X(t)=M_Y(t)$ in a neighborhood of 0, then $X\overset{d}{=}Y$. This is why matching MGFs proves distributional results.

## Probability generating functions

For **non-negative integer-valued** RVs, the **probability generating function (PGF)** is:

$$
G_X(z) = E[z^X] = \sum_{k=0}^{\infty} P(X=k)\,z^k
$$

The PGF encodes the PMF as coefficients. Derivatives give factorial moments: $G_X^{(k)}(1) = E[X(X-1)\cdots(X-k+1)]$.

```python
import numpy as np
from scipy import stats

# Poisson PGF: G(z) = exp(lambda*(z-1))
lam = 4.0

# Verify: G'(1) = E[X] = lambda
# G'(z) = lambda * exp(lambda*(z-1))  =>  G'(1) = lambda
print(f"Poisson PGF:")
print(f"  G'(1) = lambda = {lam:.4f}  (E[X])")

# G''(1) = E[X(X-1)] = lambda^2
print(f"  G''(1) = lambda^2 = {lam**2:.4f}  (E[X(X-1)])")
print(f"  => E[X^2] = G''(1) + G'(1) = {lam**2 + lam:.4f}")
print(f"  => Var[X] = E[X^2] - (E[X])^2 = {lam**2 + lam - lam**2:.4f}")

# Verify with scipy
d = stats.poisson(lam)
print(f"\n  scipy mean={d.mean():.4f}  var={d.var():.4f}")
```

```output
Poisson PGF:
  G'(1) = lambda = 4.0000  (E[X])
  G''(1) = lambda^2 = 16.0000  (E[X(X-1)])
  => E[X^2] = G''(1) + G'(1) = 20.0000
  => Var[X] = E[X^2] - (E[X])^2 = 4.0000

  scipy mean=4.0000  var=4.0000
```

## Cumulant generating function

The **cumulant generating function (CGF)** is:

$$
K_X(t) = \log M_X(t)
$$

Its derivatives at $t=0$ give **cumulants** $\kappa_k$:
- $\kappa_1 = \mu$ (mean)
- $\kappa_2 = \sigma^2$ (variance)
- $\kappa_3 = E[(X-\mu)^3]$ (related to skewness: $\gamma_1 = \kappa_3/\kappa_2^{3/2}$)
- $\kappa_4$ (related to excess kurtosis: $\gamma_2 = \kappa_4/\kappa_2^2$)

Cumulants are additive for independent RVs: $K_{X+Y}(t) = K_X(t) + K_Y(t)$.

```python
import numpy as np
from scipy import stats

# Normal CGF: K(t) = mu*t + sigma^2*t^2/2
# => kappa_1 = mu, kappa_2 = sigma^2, kappa_3 = 0, kappa_4 = 0
mu, sigma = 2.0, 3.0
print("Normal N(2, 9):")
print(f"  kappa_1 (mean)     = {mu}")
print(f"  kappa_2 (variance) = {sigma**2}")
print(f"  kappa_3 (skew*var^1.5) = 0  (symmetric)")
print(f"  kappa_4 (excess kurt*var^2) = 0  (mesokurtic)")

# Poisson: CGF is lambda*(e^t - 1)
# kappa_k = lambda for all k  (all cumulants equal lambda!)
lam = 5.0
print(f"\nPoisson(lambda={lam}):")
d = stats.poisson(lam)
sample = d.rvs(100_000, random_state=0)
print(f"  mean     = {sample.mean():.4f}  (true: {lam})")
print(f"  variance = {sample.var():.4f}   (true: {lam})")
print(f"  skewness = {stats.skew(sample):.4f}  (true: {1/np.sqrt(lam):.4f})")
```

```output
Normal N(2, 9):
  kappa_1 (mean)     = 2.0
  kappa_2 (variance) = 9
  kappa_3 (skew*var^1.5) = 0  (symmetric)
  kappa_4 (excess kurt*var^2) = 0  (mesokurtic)

Poisson(lambda=5.0):
  mean     = 4.9989  (true: 5.0)
  variance = 5.0035   (true: 5.0)
  skewness = 0.4490  (true: 0.4472)
```

The Poisson has mean = variance = $\lambda$. This **equidispersion** property is testable and often violated in real count data (negative binomial handles overdispersion).

## Try it Yourself

```python
import numpy as np

# Verify that sum of independent Normals has the expected MGF
rng = np.random.default_rng(42)
n = 500_000

mu1, s1 = 1.0, 1.5
mu2, s2 = 2.0, 2.0
X = rng.normal(mu1, s1, n)
Y = rng.normal(mu2, s2, n)
Z = X + Y  # should be N(mu1+mu2, s1^2+s2^2)

t = 0.2
mgf_Z_mc       = np.mean(np.exp(t * Z))
mgf_Z_analytic = np.exp((mu1+mu2)*t + 0.5*(s1**2+s2**2)*t**2)

print(f"Z = X + Y ~ N({mu1+mu2}, {s1**2+s2**2})")
print(f"M_Z({t}) Monte Carlo = {mgf_Z_mc:.6f}")
print(f"M_Z({t}) Analytic    = {mgf_Z_analytic:.6f}")
print(f"Sample mean of Z: {Z.mean():.4f}  (true: {mu1+mu2})")
print(f"Sample std  of Z: {Z.std():.4f}   (true: {np.sqrt(s1**2+s2**2):.4f})")
```

```output
Z = X + Y ~ N(3.0, 6.25)
M_Z(0.2) Monte Carlo = 1.896694
M_Z(0.2) Analytic    = 1.896481
Sample mean of Z: 2.9989  (true: 3.0)
Sample std  of Z: 2.5003   (true: 2.5000)
```

## Exercise

Show that the sum of two independent Poisson RVs is Poisson. (1) Write the CGF of $X\sim\text{Pois}(\lambda_1)$ and $Y\sim\text{Pois}(\lambda_2)$. (2) Use additivity of CGFs to find $K_{X+Y}$. (3) What distribution does this CGF identify? (4) Verify by simulation.

<details>
<summary>Solution</summary>

```python
import numpy as np
from scipy import stats

lam1, lam2 = 3.0, 5.0
# CGF of Pois(lam): K(t) = lam*(e^t - 1)
# K_{X+Y}(t) = lam1*(e^t-1) + lam2*(e^t-1) = (lam1+lam2)*(e^t-1)
# => X+Y ~ Pois(lam1+lam2)
print(f"X ~ Pois({lam1}), Y ~ Pois({lam2})")
print(f"K_X(t) + K_Y(t) = ({lam1}+{lam2})*(e^t - 1) => X+Y ~ Pois({lam1+lam2})")

rng = np.random.default_rng(5)
X = rng.poisson(lam1, 100_000)
Y = rng.poisson(lam2, 100_000)
Z = X + Y
Z_direct = rng.poisson(lam1+lam2, 100_000)

print(f"\nZ = X+Y:      mean={Z.mean():.4f}  var={Z.var():.4f}")
print(f"Pois({lam1+lam2}): mean={Z_direct.mean():.4f}  var={Z_direct.var():.4f}")
print(f"True: mean=var={lam1+lam2}")

# KS test: does Z follow Pois(8)?
ks = stats.kstest(Z, stats.poisson(lam1+lam2).cdf)
print(f"KS test p-value: {ks.pvalue:.4f}  (high = consistent with Pois(8))")
```

$K_X(t)+K_Y(t) = (\lambda_1+\lambda_2)(e^t-1)$, which is the CGF of $\text{Pois}(\lambda_1+\lambda_2)$.
</details>
