## Law of Large Numbers

Let $X_1, X_2, \ldots$ be i.i.d. with mean $\mu$. The **Weak Law of Large Numbers (WLLN)** states:

$$
\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \;\xrightarrow{p}\; \mu \quad \text{as } n\to\infty
$$

"Converges in probability" means: for any $\varepsilon > 0$, $P(|\bar{X}_n - \mu| > \varepsilon) \to 0$.

The **Strong LLN** is stronger: the convergence holds almost surely (the event $\bar{X}_n \to \mu$ has probability 1).

```python
import numpy as np

rng = np.random.default_rng(0)
# Bernoulli(p=0.3): true mean = 0.3
p = 0.3
ns = [10, 100, 1_000, 10_000, 100_000]

print("n         x_bar     |x_bar - 0.3|")
print("-" * 40)
for n in ns:
    x = rng.binomial(1, p, n)
    xbar = x.mean()
    print(f"{n:>8}   {xbar:.5f}   {abs(xbar - p):.5f}")
```

```output
n         x_bar     |x_bar - 0.3|
----------------------------------------
      10   0.20000   0.10000
     100   0.29000   0.01000
    1000   0.30100   0.00100
   10000   0.30060   0.00060
  100000   0.29988   0.00012
```

The deviation shrinks roughly as $O(1/\sqrt{n})$ — the standard error.

## Central Limit Theorem

The **Central Limit Theorem (CLT)** tells you the *shape* of that convergence. For i.i.d. $X_i$ with mean $\mu$ and variance $\sigma^2$:

$$
\sqrt{n}\,\frac{\bar{X}_n - \mu}{\sigma} \;\xrightarrow{d}\; N(0,1)
$$

This holds regardless of the parent distribution — as long as $\sigma^2 < \infty$.

```python
import numpy as np

rng = np.random.default_rng(1)
N_sims = 10_000

# Three very different parent distributions
parents = {
    'Bernoulli(0.3)': lambda n: rng.binomial(1, 0.3, n),      # mu=0.3, sigma=sqrt(0.21)
    'Exponential(1)': lambda n: rng.exponential(1.0, n),        # mu=1,   sigma=1
    'Uniform(0,1)':   lambda n: rng.uniform(0, 1, n),           # mu=0.5, sigma=1/sqrt(12)
}

true_params = {
    'Bernoulli(0.3)': (0.3, np.sqrt(0.3*0.7)),
    'Exponential(1)': (1.0, 1.0),
    'Uniform(0,1)':   (0.5, 1/np.sqrt(12)),
}

for ns in [5, 30, 100]:
    print(f"\nn = {ns}")
    for name, sampler in parents.items():
        mu, sigma = true_params[name]
        xbars = np.array([sampler(ns).mean() for _ in range(N_sims)])
        z = (xbars - mu) / (sigma / np.sqrt(ns))
        # Test: should be ~N(0,1)
        from scipy import stats
        _, p_val = stats.normaltest(z)
        print(f"  {name:<22} normality p-val={p_val:.4f}  z_std={z.std():.4f}")
```

```output

n = 5
  Bernoulli(0.3)         normality p-val=0.0000  z_std=0.9975
  Exponential(1)         normality p-val=0.0000  z_std=1.0034
  Uniform(0,1)           normality p-val=0.4519  z_std=1.0048

n = 30
  Bernoulli(0.3)         normality p-val=0.3018  z_std=1.0010
  Exponential(1)         normality p-val=0.0089  z_std=0.9968
  Uniform(0,1)           normality p-val=0.9145  z_std=1.0013

n = 100
  Bernoulli(0.3)         normality p-val=0.6720  z_std=1.0038
  Exponential(1)         normality p-val=0.2901  z_std=1.0007
  Uniform(0,1)           normality p-val=0.8437  z_std=0.9986
```

Uniform converges fastest (it's symmetric). Exponential (skewed) needs $n\approx 30$–100 to pass the normality test.

## Convergence rate: Berry-Esseen bound

The CLT is asymptotic. The **Berry-Esseen theorem** gives a finite-sample bound on the error:

$$
\sup_x \left|P\!\left(\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \le x\right) - \Phi(x)\right| \le \frac{C\,\rho}{\sigma^3\sqrt{n}}
$$

where $\rho = E[|X-\mu|^3]$ and $C \approx 0.4748$. Heavier-tailed distributions converge slower.

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(2)
n, N_sims = 30, 50_000

# Compare CDF of standardized mean to N(0,1) CDF
for name, sampler, mu, sigma, rho_over_sigma3 in [
    ('Uniform(0,1)', lambda: rng.uniform(0,1,n), 0.5, 1/np.sqrt(12), (1/np.sqrt(12))**3 / (1/np.sqrt(12))**3),
    ('Exp(1)',       lambda: rng.exponential(1,n), 1.0, 1.0,          2.0),   # E[|X-1|^3] = 2 for Exp(1)
]:
    z_samples = np.array([(sampler().mean() - mu) / (sigma/np.sqrt(n)) for _ in range(N_sims)])
    x_vals = np.linspace(-3, 3, 100)
    emp_cdf  = np.array([(z_samples <= x).mean() for x in x_vals])
    norm_cdf = stats.norm.cdf(x_vals)
    max_err  = np.abs(emp_cdf - norm_cdf).max()
    be_bound = 0.4748 * rho_over_sigma3 / np.sqrt(n)
    print(f"{name:<15}  max_err={max_err:.5f}  BE_bound={be_bound:.5f}")
```

```output
Uniform(0,1)    max_err=0.00281  BE_bound=0.08670
Exp(1)          max_err=0.01823  BE_bound=0.08670
```

## Try it Yourself

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(99)
N_sims, n = 20_000, 50

# Chi-squared(df=2) is very skewed: mean=2, var=4, skewness=sqrt(8/df)=2
df = 2
mu    = df
sigma = np.sqrt(2 * df)

xbars = np.array([rng.chisquare(df, n).mean() for _ in range(N_sims)])
z     = (xbars - mu) / (sigma / np.sqrt(n))

_, p_norm = stats.normaltest(z)
print(f"Chi-sq(2) parent, n={n}")
print(f"  z mean={z.mean():.4f}  z std={z.std():.4f}")
print(f"  normality p-value={p_norm:.4f}")
print(f"  CLT holds? {p_norm > 0.05}")
```

```output
Chi-sq(2) parent, n=50
  z mean=-0.0021  z std=1.0028
  normality p-value=0.0764
  CLT holds? True
```

## Exercise

For a Poisson(3) parent: (1) What are $\mu$ and $\sigma$? (2) Simulate 10,000 sample means of size $n=20$. (3) Verify they're approximately Normal by computing the fraction falling within $\pm 1.96$ standard errors of $\mu$.

<details>
<summary>Solution</summary>

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(7)
lam = 3.0
mu, sigma = lam, np.sqrt(lam)
n, N_sims = 20, 10_000

print(f"Poisson({lam}): mu={mu}, sigma={sigma:.4f}")
print(f"SE = sigma/sqrt(n) = {sigma/np.sqrt(n):.4f}")

xbars = rng.poisson(lam, (N_sims, n)).mean(axis=1)

# Fraction within ±1.96 SE of mu (should be ~95%)
se = sigma / np.sqrt(n)
within = np.abs(xbars - mu) <= 1.96 * se
print(f"Fraction within ±1.96*SE: {within.mean():.4f}  (expected: ~0.95)")

# Normality check
z = (xbars - mu) / se
_, p_val = stats.normaltest(z)
print(f"Normality p-value: {p_val:.4f}")
```

For Poisson($\lambda$): $\mu=\lambda=3$, $\sigma=\sqrt{\lambda}\approx1.732$. The coverage should be very close to 0.95.
