## Random variables map outcomes to numbers

A **random variable** $X$ is a function from $\Omega$ to $\mathbb{R}$. You don't care about
the sample space itself — you care about which numbers $X$ can take and with what probability.
A coin flip is boring; the number of heads in 10 flips is a random variable that actually
tells you something.

**Discrete:** $X$ takes countably many values. Described by its **probability mass function** (PMF):
$p(x) = P(X = x)$, which must satisfy $p(x) \ge 0$ and $\sum_x p(x) = 1$.

**Continuous:** $X$ takes values in an interval. Described by its **probability density function** (PDF) $f(x)$, where $P(a \le X \le b) = \int_a^b f(x)\,dx$. The PDF itself is not a probability — only integrals of it are.

## Expectation — the center of mass

$$
E[X] = \begin{cases} \sum_x x \, p(x) & \text{discrete} \\ \int_{-\infty}^\infty x \, f(x)\,dx & \text{continuous} \end{cases}
$$

Key properties: **linearity** ($E[aX + b] = aE[X] + b$) and **linearity of sums**
($E[X + Y] = E[X] + E[Y]$, no independence required).

```python
import numpy as np

rng = np.random.default_rng(1)
N = 500_000

# Fair die: X ~ Uniform{1,...,6}, true E[X] = 3.5
rolls = rng.integers(1, 7, N).astype(np.float64)
print(f"E[X] sample = {rolls.mean():.4f}   true = 3.5")

# E[g(X)] = sum g(x) p(x) — law of the unconscious statistician (LOTUS)
# E[X^2] for a fair die = (1+4+9+16+25+36)/6
print(f"E[X^2] sample = {(rolls**2).mean():.4f}   true = {(1+4+9+16+25+36)/6:.4f}")
```

```output
E[X] sample = 3.4994   true = 3.5
E[X^2] sample = 15.1685   true = 15.1667
```

## Variance — the spread

$$
\text{Var}(X) = E[(X - \mu)^2] = E[X^2] - (E[X])^2, \quad \mu = E[X]
$$

The second form (computational) is usually easier. The **standard deviation** $\sigma = \sqrt{\text{Var}(X)}$ lives in the same units as $X$.

Key properties: $\text{Var}(aX + b) = a^2 \text{Var}(X)$. For **independent** $X, Y$:
$\text{Var}(X + Y) = \text{Var}(X) + \text{Var}(Y)$. Independence is required here —
this breaks down when $X$ and $Y$ are correlated (ch. 5).

```python
import numpy as np

rng = np.random.default_rng(1)
N = 500_000
rolls = rng.integers(1, 7, N).astype(np.float64)

mu     = rolls.mean()
var    = rolls.var()          # population variance (divides by N)
var_df = rolls.var(ddof=1)    # sample variance (divides by N-1, unbiased)

true_var = sum((x - 3.5)**2 * (1/6) for x in range(1, 7))
print(f"sample mean = {mu:.4f}    true = 3.5000")
print(f"pop var     = {var:.4f}    true = {true_var:.4f}")
print(f"sample var  = {var_df:.4f}  (ddof=1, unbiased for large N)")
```

```output
sample mean = 3.4994    true = 3.5000
pop var     = 2.9148    true = 2.9167
sample var  = 2.9148  (ddof=1, unbiased for large N)
```

## Moments — the shape descriptors

The $k$-th **raw moment** of $X$ is $E[X^k]$. The $k$-th **central moment** is
$E[(X - \mu)^k]$. The first four matter most:

| $k$ | Name | Formula |
|-----|------|---------|
| 1 | Mean | $\mu = E[X]$ |
| 2 | Variance | $E[(X-\mu)^2]$ |
| 3 | Skewness | $E[(X-\mu)^3] / \sigma^3$ |
| 4 | Excess kurtosis | $E[(X-\mu)^4] / \sigma^4 - 3$ |

**Skewness** measures asymmetry: positive = right tail heavier (income distributions);
negative = left tail heavier (exam scores near a ceiling). **Excess kurtosis** measures
tail weight relative to a Normal distribution (which has kurtosis 0 by this definition).

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(2)

# Right-skewed: lognormal
x_log = rng.lognormal(mean=0.0, sigma=0.8, size=100_000)
# Symmetric: normal
x_norm = rng.normal(0, 1, 100_000)

for name, x in [('lognormal', x_log), ('normal', x_norm)]:
    sk = stats.skew(x)
    ku = stats.kurtosis(x)   # excess kurtosis
    print(f"{name:10s}  skew={sk:+.3f}  excess_kurtosis={ku:+.3f}")
```

```output
lognormal   skew=+2.419  excess_kurtosis=+10.471
normal      skew=+0.003  excess_kurtosis=-0.007
```

The lognormal is strongly right-skewed and heavy-tailed. Most financial returns and
natural measurements are lognormal — the Normal assumption is almost always wrong for them.

## CDF — the full distribution in one function

The **cumulative distribution function** (CDF) $F(x) = P(X \le x)$ encodes the entire
distribution. It is always non-decreasing, right-continuous, and ranges from 0 to 1.

$$
F(x) = \begin{cases} \sum_{t \le x} p(t) & \text{discrete} \\ \int_{-\infty}^x f(t)\,dt & \text{continuous} \end{cases}
$$

For a continuous RV, $f(x) = F'(x)$. The **quantile function** $F^{-1}(q)$ is its inverse:
it tells you the value $x$ such that $P(X \le x) = q$. Quantiles are how you compute
confidence intervals and p-values.

## Try it Yourself

```python
import numpy as np

rng = np.random.default_rng(5)
N = 300_000

# Exponential(lambda=2): E[X] = 1/lambda = 0.5, Var[X] = 1/lambda^2 = 0.25
lam = 2.0
x = rng.exponential(scale=1/lam, size=N)

print(f"E[X]    sample={x.mean():.4f}   true={1/lam:.4f}")
print(f"Var[X]  sample={x.var():.4f}   true={1/lam**2:.4f}")
print(f"Std[X]  sample={x.std():.4f}   true={1/lam:.4f}")

# Empirical CDF at x=0.5 (true: 1 - exp(-2*0.5) = 1 - e^{-1} ≈ 0.6321)
ecdf_at_half = (x <= 0.5).mean()
import math
true_cdf = 1 - math.exp(-lam * 0.5)
print(f"P(X<=0.5) sample={ecdf_at_half:.4f}  true={true_cdf:.4f}")
```

```output
E[X]    sample=0.5003   true=0.5000
Var[X]  sample=0.2500   true=0.2500
Std[X]  sample=0.5000   true=0.5000
P(X<=0.5) sample=0.6322  true=0.6321
```

## Exercise

For $X \sim \text{Uniform}(a, b)$: derive $E[X]$ and $\text{Var}(X)$ from the PDF
$f(x) = 1/(b-a)$ on $[a, b]$. Then verify numerically with $a=2$, $b=8$.

<details>
<summary>Solution</summary>

$$
E[X] = \int_a^b \frac{x}{b-a}\,dx = \frac{a+b}{2}
$$
$$
E[X^2] = \int_a^b \frac{x^2}{b-a}\,dx = \frac{a^2+ab+b^2}{3}
$$
$$
\text{Var}(X) = E[X^2] - (E[X])^2 = \frac{(b-a)^2}{12}
$$

```python
import numpy as np

rng = np.random.default_rng(0)
a, b = 2.0, 8.0
x = rng.uniform(a, b, 500_000)

print(f"E[X]     sample={x.mean():.4f}   true={(a+b)/2:.4f}")
print(f"Var(X)   sample={x.var():.4f}   true={(b-a)**2/12:.4f}")
```

For Uniform$(a,b)$: mean is the midpoint, variance scales with the square of the range.
</details>
