## The Normal distribution

The Normal (Gaussian) distribution is parameterized by mean $\mu$ and standard deviation $\sigma$:

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}}\exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)
$$

The **Z-score** $Z = (X-\mu)/\sigma$ standardizes to $N(0,1)$. The 68-95-99.7 rule:
- 68% of mass within $\pm 1\sigma$
- 95% within $\pm 2\sigma$
- 99.7% within $\pm 3\sigma$

```python
import numpy as np
from scipy import stats

mu, sigma = 5.0, 2.0
dist = stats.norm(loc=mu, scale=sigma)

print(f"mean={dist.mean():.1f}  var={dist.var():.1f}")
print(f"P(|X-mu| < 1*sigma) = {dist.cdf(mu+sigma) - dist.cdf(mu-sigma):.4f}")
print(f"P(|X-mu| < 2*sigma) = {dist.cdf(mu+2*sigma) - dist.cdf(mu-2*sigma):.4f}")
print(f"P(|X-mu| < 3*sigma) = {dist.cdf(mu+3*sigma) - dist.cdf(mu-3*sigma):.4f}")

rng = np.random.default_rng(0)
sample = rng.normal(mu, sigma, 10_000)
print(f"\nSample mean={sample.mean():.3f}  std={sample.std():.3f}")
```

```output
mean=5.0  var=4.0
P(|X-mu| < 1*sigma) = 0.6827
P(|X-mu| < 2*sigma) = 0.9545
P(|X-mu| < 3*sigma) = 0.9973

Sample mean=5.007  std=2.001
```

The Normal is closed under linear transformations: if $X\sim N(\mu,\sigma^2)$ then $aX+b\sim N(a\mu+b,a^2\sigma^2)$. The sum of independent Normals is Normal.

## The Exponential distribution

The Exponential distribution models the time until the first event in a Poisson process with rate $\lambda$:

$$
f(x) = \lambda e^{-\lambda x}, \quad x \ge 0
$$

Mean $= 1/\lambda$, variance $= 1/\lambda^2$.

The **memoryless property** is its defining feature: $P(X > s + t \mid X > s) = P(X > t)$. If you've already waited $s$ minutes, the remaining wait has the same distribution as the original.

```python
import numpy as np
from scipy import stats

lam = 0.5  # average 2 minutes between events
dist = stats.expon(scale=1/lam)

print(f"mean={dist.mean():.1f}  std={dist.std():.1f}")

# Memoryless: P(X > 3+2 | X > 3) should equal P(X > 2)
s, t = 3.0, 2.0
p_gt_s = 1 - dist.cdf(s)
p_gt_st = 1 - dist.cdf(s + t)
conditional = p_gt_st / p_gt_s
p_gt_t = 1 - dist.cdf(t)

print(f"P(X > {s+t} | X > {s}) = {conditional:.4f}")
print(f"P(X > {t})              = {p_gt_t:.4f}")
print(f"Memoryless: {np.isclose(conditional, p_gt_t)}")
```

```output
mean=2.0  std=2.0
P(X > 5.0 | X > 3) = 0.3679
P(X > 2.0)              = 0.3679
Memoryless: True
```

## The Gamma distribution

The Gamma distribution generalizes the Exponential. With shape $\alpha$ and rate $\beta$:

$$
f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}, \quad x > 0
$$

Mean $= \alpha/\beta$, variance $= \alpha/\beta^2$.

Special cases:
- $\alpha=1$: Exponential with rate $\beta$
- $\alpha=k/2, \beta=1/2$: Chi-squared with $k$ degrees of freedom
- Integer $\alpha$: Erlang distribution (sum of $\alpha$ independent exponentials)

```python
import numpy as np
from scipy import stats

# Verify: sum of alpha independent Exp(beta) ~ Gamma(alpha, beta)
alpha, beta = 4, 0.5
rng = np.random.default_rng(1)
n = 50_000

# Direct Gamma samples
gamma_samples = rng.gamma(shape=alpha, scale=1/beta, size=n)

# Sum of alpha Exponentials
exp_samples = rng.exponential(scale=1/beta, size=(n, alpha)).sum(axis=1)

print(f"Gamma direct:  mean={gamma_samples.mean():.3f}  std={gamma_samples.std():.3f}")
print(f"Sum of Exp:    mean={exp_samples.mean():.3f}  std={exp_samples.std():.3f}")
print(f"Theoretical:   mean={alpha/beta:.3f}  std={np.sqrt(alpha)/beta:.3f}")
```

```output
Gamma direct:  mean=7.995  std=3.999
Sum of Exp:    mean=8.001  std=4.001
Theoretical:   mean=8.000  std=4.000
```

## The Beta distribution

The Beta distribution has support $[0,1]$, making it a natural prior for probabilities:

$$
f(x) = \frac{x^{\alpha-1}(1-x)^{\beta-1}}{B(\alpha,\beta)}, \quad 0 \le x \le 1
$$

Mean $= \alpha/(\alpha+\beta)$. The shape parameters control skew:
- $\alpha=\beta=1$: Uniform
- $\alpha=\beta > 1$: Symmetric, bell-shaped around 0.5
- $\alpha < \beta$: Left-skewed (mass near 0)
- $\alpha > \beta$: Right-skewed (mass near 1)

```python
import numpy as np
from scipy import stats

configs = [
    (1, 1, "Uniform"),
    (2, 5, "skewed left"),
    (5, 2, "skewed right"),
    (5, 5, "symmetric around 0.5"),
]

for a, b, desc in configs:
    d = stats.beta(a, b)
    print(f"Beta({a},{b}) [{desc}]: mean={d.mean():.3f}  var={d.var():.4f}")
```

```output
Beta(1,1) [Uniform]: mean=0.500  var=0.0833
Beta(2,5) [skewed left]: mean=0.286  var=0.0255
Beta(5,2) [skewed right]: mean=0.714  var=0.0255
Beta(5,5) [symmetric around 0.5]: mean=0.500  var=0.0208
```

The Beta is the **conjugate prior** for the Bernoulli/Binomial likelihood. After observing $k$ successes in $n$ trials, a $\text{Beta}(\alpha, \beta)$ prior becomes $\text{Beta}(\alpha+k, \beta+n-k)$.

```python
import numpy as np
from scipy import stats

# Bayesian coin-flipping: start with Beta(1,1) = Uniform prior
# Observe 7 heads in 10 flips
alpha_prior, beta_prior = 1, 1
k, n = 7, 10

alpha_post = alpha_prior + k
beta_post  = beta_prior + (n - k)

posterior = stats.beta(alpha_post, beta_post)
print(f"Posterior: Beta({alpha_post}, {beta_post})")
print(f"  mean = {posterior.mean():.4f}  (MLE = {k/n:.4f})")
print(f"  95% credible interval: [{posterior.ppf(0.025):.3f}, {posterior.ppf(0.975):.3f}]")
```

```output
Posterior: Beta(8, 4)
  mean = 0.6667  (MLE = 0.7000)
  95% credible interval: [0.355, 0.901]
```

The posterior mean 0.667 pulls the MLE (0.7) toward the prior mean (0.5) — the Bayesian shrinkage effect.

## Try it Yourself

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(42)
n = 10_000

# Generate data from a mixture: 60% Normal(0,1) + 40% Exponential(rate=2)
normal_part = rng.normal(0, 1, int(0.6 * n))
exp_part    = rng.exponential(scale=0.5, size=int(0.4 * n))
data = np.concatenate([normal_part, exp_part])

print(f"Sample mean:     {data.mean():.4f}")
print(f"Sample std:      {data.std():.4f}")
print(f"Sample skewness: {stats.skew(data):.4f}")
print(f"Sample kurtosis: {stats.kurtosis(data):.4f}")

# What fraction is negative? (only the Normal part can be negative)
print(f"Fraction < 0: {(data < 0).mean():.4f}")
```

```output
Sample mean:     0.1970
Sample std:      0.9658
Sample skewness: 0.5223
Sample kurtosis: 0.7498
Fraction < 0: 0.2974
```

## Exercise

You observe 200 waiting times (in minutes) and suspect they follow $\text{Exponential}(\lambda)$. The sample mean is 4.2 minutes. (1) What is the MLE of $\lambda$? (2) Using `scipy.stats.kstest`, test whether a sample from this estimated exponential fits the data. (3) What is $P(X > 10)$ under this model?

<details>
<summary>Solution</summary>

```python
import numpy as np
from scipy import stats

rng = np.random.default_rng(7)
# Simulate the scenario
true_lam = 1/4.2
data = rng.exponential(scale=1/true_lam, size=200)

# 1) MLE: lambda_hat = 1 / x_bar
xbar = data.mean()
lam_hat = 1 / xbar
print(f"Sample mean: {xbar:.4f}  MLE lambda: {lam_hat:.4f}")

# 2) KS test
ks_stat, p_val = stats.kstest(data, 'expon', args=(0, 1/lam_hat))
print(f"KS statistic: {ks_stat:.4f}  p-value: {p_val:.4f}")
print(f"Reject H0 (not Exponential)? {p_val < 0.05}")

# 3) P(X > 10)
dist = stats.expon(scale=1/lam_hat)
print(f"P(X > 10) = {1 - dist.cdf(10):.4f}")
```

The MLE of $\lambda$ for the Exponential is $\hat\lambda = 1/\bar{x}$. A p-value above 0.05 on the KS test means we fail to reject the Exponential assumption.
</details>
