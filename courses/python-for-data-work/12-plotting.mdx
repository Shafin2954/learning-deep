## Three layers of the Python plotting stack

Python has three mainstream plotting libraries, each building on (or alongside) the previous:

| Library | Level | When to use |
|---------|-------|-------------|
| **matplotlib** | low-level, full control | custom figures, publication plots, embedding in larger systems |
| **seaborn** | statistical, wraps matplotlib | quick distribution/relationship plots, themes |
| **plotly** | interactive, browser-based | dashboards, exploratory analysis you'll share |

You need all three. Seaborn calls matplotlib internally, so matplotlib mastery unlocks
everything.

## matplotlib anatomy: Figure, Axes, Artists

Every matplotlib plot is built from:
- **Figure** — the canvas. Has size (`figsize`), DPI, and one or more Axes.
- **Axes** (`ax`) — one plot panel. Has x/y axes, title, labels, gridlines, and the actual
  data renderers.
- **Artists** — everything drawn: lines, bars, patches, text, images.

```python
import matplotlib.pyplot as plt
import numpy as np

# The object-oriented API (use this — not plt.plot() shortcuts)
fig, ax = plt.subplots(figsize=(6, 3))

x = np.linspace(0, 2 * np.pi, 200)
ax.plot(x, np.sin(x), label='sin(x)', color='steelblue', linewidth=2)
ax.plot(x, np.cos(x), label='cos(x)', color='tomato',    linestyle='--')

ax.set_title('Sine and cosine')
ax.set_xlabel('x (radians)')
ax.set_ylabel('amplitude')
ax.legend(loc='upper right')
ax.grid(alpha=0.3)

fig.tight_layout()
plt.show()
```

```output
[Figure: sine and cosine curves on clean axes with legend]
```

Always use the object-oriented API (`fig, ax = plt.subplots()`), not `plt.plot()` /
`plt.xlabel()`. The latter mutates a global state that breaks the moment you have two
subplots. The OO API is explicit and predictable.

### Subplots — multiple panels

```python
import matplotlib.pyplot as plt
import numpy as np

fig, axes = plt.subplots(1, 3, figsize=(12, 3), sharey=True)

x = np.linspace(0, 4, 100)
fns  = [np.sin, np.cos, lambda t: np.sin(t) * np.exp(-t/3)]
names = ['sin', 'cos', 'damped sin']

for ax, fn, name in zip(axes, fns, names):
    ax.plot(x, fn(x), linewidth=2)
    ax.set_title(name)
    ax.grid(alpha=0.3)

fig.tight_layout()
plt.show()
```

```output
[Figure: three side-by-side plots sharing the y-axis scale]
```

`sharey=True` locks the y-axis range across all three panels so comparisons are honest.
`fig.tight_layout()` prevents labels from overlapping.

## seaborn: statistical plots with one call

seaborn wraps matplotlib and understands pandas DataFrames directly. It adds:
- Automatic grouping by a `hue` column.
- Built-in themes (`sns.set_theme()`).
- Distribution plots that would be tedious in raw matplotlib.

```python
import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

sns.set_theme(style='whitegrid', palette='muted')

rng = np.random.default_rng(5)
df = pd.DataFrame({
    'group':  rng.choice(['A', 'B', 'C'], 300),
    'value':  np.concatenate([
                  rng.normal(10, 2, 100),
                  rng.normal(14, 3, 100),
                  rng.normal(12, 1.5, 100),
              ]),
})

fig, axes = plt.subplots(1, 2, figsize=(10, 4))

# Distribution per group
sns.histplot(data=df, x='value', hue='group', bins=20, ax=axes[0], alpha=0.7)
axes[0].set_title('Histograms by group')

# Box plot
sns.boxplot(data=df, x='group', y='value', ax=axes[1])
axes[1].set_title('Box plot by group')

fig.tight_layout()
plt.show()
```

```output
[Figure: histogram with three overlapping distributions and a box plot showing medians and spread per group]
```

### Scatter with regression line

```python
import seaborn as sns
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

sns.set_theme(style='whitegrid')
rng = np.random.default_rng(0)
n = 100
x = rng.uniform(0, 10, n)
y = 2 * x + rng.normal(0, 3, n)
df = pd.DataFrame({'x': x, 'y': y})

fig, ax = plt.subplots(figsize=(5, 4))
sns.regplot(data=df, x='x', y='y', ax=ax, scatter_kws={'alpha': 0.5})
ax.set_title('Linear relationship with regression line')
plt.show()
```

```output
[Figure: scatter plot of ~100 points with a fitted regression line and confidence band]
```

## plotly: interactive plots

plotly Express (`px`) is plotly's high-level API. The output is an interactive HTML widget
that renders in a browser or Jupyter notebook (the code-runner displays it inline).

```python
import plotly.express as px
import pandas as pd
import numpy as np

rng = np.random.default_rng(3)
N = 500
df = pd.DataFrame({
    'x':      rng.standard_normal(N),
    'y':      rng.standard_normal(N),
    'size':   rng.uniform(5, 20, N),
    'group':  rng.choice(['cat', 'dog', 'bird'], N),
    'value':  rng.exponential(50, N).round(1),
})

fig = px.scatter(
    df, x='x', y='y', color='group', size='size',
    hover_data=['value'],
    title='Interactive scatter (hover for value)',
    template='plotly_white',
)
fig.show()
```

```output
[Interactive plotly figure: scatter plot with colored groups; hover tooltips show 'value'. Zoom and pan enabled.]
```

plotly charts are interactive: hover for tooltips, click legend to toggle groups, drag
to zoom, double-click to reset. This is the right tool for exploratory analysis you'll
share with others.

### When to use which

- **matplotlib** for publication-quality static figures (papers, slides), custom layouts, and
  whenever you need precise pixel control.
- **seaborn** for quick EDA plots: distributions, correlations, facets. Its `pairplot` and
  `heatmap` are hard to beat for speed.
- **plotly** when the reader needs to interact with the data — explore subsets, read exact
  values, compare scales interactively.

## Heatmap — seaborn's correlation matrix

```python
import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

rng = np.random.default_rng(7)
df = pd.DataFrame(rng.standard_normal((100, 5)), columns=list('ABCDE'))
df['B'] = df['A'] * 0.8 + rng.standard_normal(100) * 0.3   # correlate B with A

corr = df.corr()
fig, ax = plt.subplots(figsize=(5, 4))
sns.heatmap(corr, annot=True, fmt='.2f', cmap='coolwarm', center=0, ax=ax)
ax.set_title('Correlation matrix')
fig.tight_layout()
plt.show()
```

```output
[Figure: 5×5 heatmap with annotated correlation values; A-B correlation ≈ 0.93; others near 0]
```

## Try it Yourself

```python
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
import pandas as pd

sns.set_theme(style='whitegrid')
rng = np.random.default_rng(11)

# Build a small time series with two groups
idx = pd.date_range('2023-01-01', periods=60, freq='D')
df = pd.DataFrame({
    'date':    list(idx) * 2,
    'group':   ['product_A']*60 + ['product_B']*60,
    'revenue': np.concatenate([
        50 + 0.5 * np.arange(60) + rng.normal(0, 5, 60),
        70 + 0.2 * np.arange(60) + rng.normal(0, 8, 60),
    ]).round(1),
})

fig, axes = plt.subplots(1, 2, figsize=(12, 4))

# 1) Line plot by group (matplotlib)
for grp, sub in df.groupby('group'):
    axes[0].plot(sub['date'], sub['revenue'], label=grp, alpha=0.8)
axes[0].set_title('Daily revenue by product')
axes[0].legend()
axes[0].tick_params(axis='x', rotation=30)

# 2) Box plot by month using seaborn
df['month'] = df['date'].dt.strftime('%b')
month_order = ['Jan', 'Feb', 'Mar']
sns.boxplot(data=df, x='month', y='revenue', hue='group',
            order=month_order, ax=axes[1])
axes[1].set_title('Monthly revenue distribution')

fig.tight_layout()
plt.show()
```

```output
[Figure: left panel shows two rising trend lines for products A and B over 60 days; right panel shows side-by-side box plots per month and group showing revenue spread]
```

## Exercise

Load (or simulate) a DataFrame with columns `feature_1` … `feature_5` and `target` (100
rows, some correlated). Produce:

1. A seaborn `pairplot` colored by `target > target.median()`.
2. A matplotlib bar chart of the correlation of each feature with `target`.

<summary>Solution</summary>

```python
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd
import numpy as np

rng = np.random.default_rng(0)
n = 100
base = rng.standard_normal(n)
df = pd.DataFrame({
    'feature_1': base + rng.normal(0, 0.3, n),
    'feature_2': -base * 0.6 + rng.normal(0, 0.5, n),
    'feature_3': rng.standard_normal(n),
    'feature_4': rng.standard_normal(n),
    'feature_5': rng.standard_normal(n),
    'target':    base * 2 + rng.normal(0, 1, n),
})
df['target_high'] = (df['target'] > df['target'].median()).map({True: 'high', False: 'low'})

# 1) Pairplot
g = sns.pairplot(df.drop('target_high', axis=1), hue=df['target_high'], corner=True, plot_kws={'alpha': 0.5})
g.fig.suptitle('Pairplot', y=1.01)
plt.show()

# 2) Correlation bar chart
corrs = df.drop(['target', 'target_high'], axis=1).corrwith(df['target']).sort_values()
fig, ax = plt.subplots(figsize=(5, 3))
colors = ['tomato' if c < 0 else 'steelblue' for c in corrs]
ax.barh(corrs.index, corrs.values, color=colors)
ax.axvline(0, color='black', linewidth=0.8)
ax.set_title('Correlation with target')
fig.tight_layout()
plt.show()
```
