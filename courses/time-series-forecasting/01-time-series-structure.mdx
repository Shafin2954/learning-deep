## What makes a time series different

A time series is a sequence of observations $y_1, y_2, \ldots, y_T$ ordered by time. The
key difference from ordinary tabular data: observations are **serially dependent** — knowing
$y_{t-1}$ tells you something about $y_t$. That dependence is the signal you want to model.

Three structural components appear in almost every real series:

$$
y_t = T_t + S_t + R_t \quad \text{(additive)}
$$

- $T_t$ — **trend**: the long-run direction (up, down, flat)
- $S_t$ — **seasonal**: a repeating pattern at a fixed frequency (weekly, monthly, yearly)
- $R_t$ — **residual**: everything left over; ideally unpredictable noise

When the seasonal amplitude grows proportionally with the level, use the **multiplicative**
form instead:

$$
y_t = T_t \times S_t \times R_t
$$

Taking logs of a multiplicative series converts it to additive: $\log y_t = \log T_t + \log S_t + \log R_t$.

## Datetime indexing in pandas

pandas carries a `DatetimeIndex` that unlocks resampling, shifting, and frequency inference.

```python
import pandas as pd
import numpy as np

# Build a monthly index for 5 years
idx = pd.date_range(start='2019-01-01', periods=60, freq='ME')
rng = np.random.default_rng(42)

# Additive series: linear trend + monthly seasonality + noise
t = np.arange(60)
trend    = 100 + 0.8 * t
seasonal = 15 * np.sin(2 * np.pi * t / 12)
noise    = rng.normal(0, 4, 60)
y = trend + seasonal + noise

series = pd.Series(y, index=idx, name='value')
print(series.head())
print(f"\nFrequency: {series.index.freq}")
print(f"Annual mean:\n{series.resample('YE').mean().round(1)}")
```

```output
2019-01-31    104.99
2019-02-28    111.82
2019-03-31    125.52
2019-04-30    130.30
2019-05-31    131.27
Freq: ME, Name: value, dtype: float64

Frequency: ME
Annual mean:
2019-12-31    118.6
2020-12-31    125.9
2021-12-31    133.6
2022-12-31    141.0
2023-12-31    148.5
Freq: YE-DEC, Name: value, dtype: float64
```

`resample('YE').mean()` collapses the series to one value per year-end — the annual averages
increase ~7 units each year, confirming the upward trend.

## Simulating additive vs multiplicative series

Seeing both structures side by side clarifies when to apply the log transform.

```python
import pandas as pd
import numpy as np

rng = np.random.default_rng(0)
idx = pd.date_range('2020-01', periods=48, freq='ME')
t   = np.arange(48)

# Additive: seasonal amplitude is constant
trend_add  = 50 + 1.2 * t
seas_add   = 10 * np.sin(2 * np.pi * t / 12)
y_add      = trend_add + seas_add + rng.normal(0, 3, 48)

# Multiplicative: seasonal amplitude scales with trend
trend_mul  = 50 + 1.2 * t
seas_mul   = 1 + 0.3 * np.sin(2 * np.pi * t / 12)  # multiplier around 1
y_mul      = trend_mul * seas_mul + rng.normal(0, 3, 48)

s_add = pd.Series(y_add, index=idx)
s_mul = pd.Series(y_mul, index=idx)

# Check: seasonal amplitude early vs late
print("Additive — early peak-to-trough:", round(y_add[:12].max() - y_add[:12].min(), 1))
print("Additive — late  peak-to-trough:", round(y_add[36:].max() - y_add[36:].min(), 1))
print("Multiplicative — early:", round(y_mul[:12].max() - y_mul[:12].min(), 1))
print("Multiplicative — late: ", round(y_mul[36:].max() - y_mul[36:].min(), 1))
```

```output
Additive — early peak-to-trough: 20.2
Additive — late  peak-to-trough: 21.8
Multiplicative — early: 22.0
Multiplicative — late:  29.4
```

The additive series keeps a roughly constant seasonal swing (~20). The multiplicative series
has a larger swing late (~29) than early (~22) because the seasonal factor scales with the
growing trend.

## Try it Yourself

```python
import pandas as pd
import numpy as np

rng = np.random.default_rng(7)
idx = pd.date_range('2021-01', periods=36, freq='ME')
t   = np.arange(36)

# Start with an additive series
amp    = 10                  # seasonal amplitude
trend  = 80 + 0.5 * t
seas   = amp * np.sin(2 * np.pi * t / 12)
noise  = rng.normal(0, 2, 36)
y      = trend + seas + noise

series = pd.Series(y, index=idx)

# Basic summary
print("Mean:", round(series.mean(), 2))
print("Std: ", round(series.std(), 2))
print("Min: ", round(series.min(), 2))
print("Max: ", round(series.max(), 2))
```

```output
Mean: 94.57
Std:  8.93
Min:  74.49
Max:  111.96
```

## Exercise

Create a multiplicative time series with 60 monthly observations where the trend grows from
100 to 200. Apply a log transform and verify that the seasonal amplitude becomes constant
after the transform.

<details>
<summary>Solution</summary>

```python
import pandas as pd
import numpy as np

rng = np.random.default_rng(1)
idx = pd.date_range('2019-01', periods=60, freq='ME')
t   = np.arange(60)

trend = 100 + (100/59) * t                    # grows from 100 to 200
seas  = 1 + 0.25 * np.sin(2 * np.pi * t / 12)
noise = np.exp(rng.normal(0, 0.05, 60))       # multiplicative noise
y = trend * seas * noise

series = pd.Series(y, index=idx)
log_series = np.log(series)

early_swing = y[:12].max() - y[:12].min()
late_swing  = y[36:].max() - y[36:].min()
log_early   = log_series[:12].max() - log_series[:12].min()
log_late    = log_series[36:].max() - log_series[36:].min()

print(f"Original  — early swing: {early_swing:.1f},  late swing: {late_swing:.1f}")
print(f"Log-trans — early swing: {log_early:.3f}, late swing: {log_late:.3f}")
```

The log-transformed series shows nearly equal early and late seasonal swings, confirming the
multiplicative structure has been made additive.
</details>
