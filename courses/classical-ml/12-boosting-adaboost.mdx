## The boosting idea

Bagging builds trees in parallel from independent bootstrap samples. Boosting builds trees **sequentially**: each tree corrects the errors of the previous ensemble. The model grows stronger with each addition.

## AdaBoost

AdaBoost trains a sequence of weak learners on **reweighted** training data. Misclassified points get higher weights, forcing the next learner to focus on them.

### Algorithm

Given binary labels $y_i \in \{-1, +1\}$ and initial uniform weights $w_i = 1/n$:

1. Fit a weak learner $h_t$ on the weighted data.
2. Compute its weighted error: $\varepsilon_t = \sum_i w_i \mathbf{1}[h_t(x_i) \ne y_i]$.
3. Compute learner weight: $\alpha_t = \frac{1}{2}\ln\frac{1 - \varepsilon_t}{\varepsilon_t}$.
4. Update sample weights: $w_i \leftarrow w_i \exp(-\alpha_t y_i h_t(x_i))$, then renormalise.
5. Final: $H(x) = \text{sign}\!\left(\sum_t \alpha_t h_t(x)\right)$.

```python
import numpy as np
from sklearn.tree import DecisionTreeClassifier

def adaboost_fit(X, y, n_estimators=50):
    n = len(y)
    w = np.ones(n) / n
    stumps, alphas = [], []
    for _ in range(n_estimators):
        stump = DecisionTreeClassifier(max_depth=1)
        stump.fit(X, y, sample_weight=w)
        pred = stump.predict(X)
        err  = np.sum(w * (pred != y)) / np.sum(w)
        err  = np.clip(err, 1e-10, 1 - 1e-10)
        alpha = 0.5 * np.log((1 - err) / err)
        w *= np.exp(-alpha * y * pred)
        w /= w.sum()
        stumps.append(stump); alphas.append(alpha)
    return stumps, alphas

def adaboost_predict(stumps, alphas, X):
    vote = sum(a * s.predict(X) for a, s in zip(alphas, stumps))
    return np.sign(vote)

from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split

X, y = make_classification(n_samples=400, n_features=2, n_redundant=0,
                             n_clusters_per_class=1, random_state=0)
y = 2*y - 1   # {-1, +1}
X_tr, X_te, y_tr, y_te = train_test_split(X, y, test_size=0.2, random_state=0)

stumps, alphas = adaboost_fit(X_tr, y_tr, n_estimators=100)
pred = adaboost_predict(stumps, alphas, X_te)
scratch_acc = np.mean(pred == y_te)
print(f"Scratch AdaBoost: {scratch_acc:.3f}")
```

```output
Scratch AdaBoost: 0.913
```

## Exponential loss interpretation

AdaBoost minimises the **exponential loss** $L(y, f) = e^{-yf(x)}$. The weight update is the gradient step that reduces this loss. This connection reveals AdaBoost as a form of gradient descent in function space.

## Gradient Boosting: the functional gradient view

Gradient boosting generalises AdaBoost to **any differentiable loss**. The key insight:

> The residuals $y - \hat{y}$ are the **negative gradient** of MSE loss. Fitting the next tree to the residuals is gradient descent in function space.

For MSE loss $L(y, f) = \frac{1}{2}(y - f)^2$: negative gradient $= y - f(x) =$ residuals.
For log-loss (classification): negative gradient $= y - \sigma(f(x))$.

### Algorithm

Given initial prediction $f_0 = \bar{y}$, for $m = 1, \ldots, M$:

1. Compute pseudo-residuals: $r_i = -\frac{\partial L(y_i, f(x_i))}{\partial f(x_i)}$
2. Fit a regression tree $h_m$ to the pseudo-residuals.
3. Update: $f \leftarrow f + \eta \cdot h_m$

```python
import numpy as np
from sklearn.tree import DecisionTreeRegressor

def gradient_boosting_fit(X, y, n_estimators=100, lr=0.1, max_depth=3):
    f = np.full(len(y), y.mean())
    trees = []
    for _ in range(n_estimators):
        residuals = y - f            # negative gradient of MSE
        tree = DecisionTreeRegressor(max_depth=max_depth, random_state=0)
        tree.fit(X, residuals)
        f += lr * tree.predict(X)
        trees.append(tree)
    return trees, y.mean()

def gradient_boosting_predict(trees, f0, X, lr=0.1):
    pred = np.full(len(X), f0)
    for tree in trees:
        pred += lr * tree.predict(X)
    return pred

from sklearn.datasets import make_regression
from sklearn.model_selection import train_test_split

X, y = make_regression(n_samples=500, n_features=10, noise=20, random_state=0)
X_tr, X_te, y_tr, y_te = train_test_split(X, y, test_size=0.2, random_state=0)

trees, f0 = gradient_boosting_fit(X_tr, y_tr, n_estimators=100)
pred = gradient_boosting_predict(trees, f0, X_te)
rmse = np.sqrt(np.mean((pred - y_te)**2))
print(f"Scratch GB RMSE: {rmse:.2f}")
```

```output
Scratch GB RMSE: 28.41
```

## Shrinkage

The learning rate $\eta \in (0,1)$ scales each tree's contribution. Smaller $\eta$ requires more trees but generalises better — it prevents any single tree from over-correcting.

## Stochastic GBM

At each step, subsample a fraction of rows (and/or columns) before fitting the tree. This:
- Reduces overfitting (adds noise, like dropout).
- Speeds up training (smaller dataset per tree).
- Decorrelates trees slightly.

```python
from sklearn.ensemble import GradientBoostingRegressor
import numpy as np

for subsample in [1.0, 0.8, 0.5]:
    gb = GradientBoostingRegressor(n_estimators=200, learning_rate=0.05,
                                    max_depth=3, subsample=subsample,
                                    random_state=0).fit(X_tr, y_tr)
    rmse = np.sqrt(np.mean((gb.predict(X_te) - y_te)**2))
    print(f"subsample={subsample}: RMSE={rmse:.2f}")
```

```output
subsample=1.0: RMSE=25.93
subsample=0.8: RMSE=25.44
subsample=0.5: RMSE=25.17
```

import numpy as np
from sklearn.datasets import make_regression
from sklearn.model_selection import train_test_split
from sklearn.ensemble import GradientBoostingRegressor

X, y = make_regression(n_samples=600, n_features=15, noise=25, random_state=1)
X_tr, X_te, y_tr, y_te = train_test_split(X, y, test_size=0.2, random_state=0)

# Compare learning rate effect
print(f"{'lr':>6} | {'n_trees':>7} | {'RMSE':>7}")

Using `make_regression` (500 samples, 20 features, noise=15), implement gradient boosting from scratch using max_depth=2 stumps. Compare RMSE at 50, 100, and 200 trees vs `sklearn.GradientBoostingRegressor`. Your RMSE should be within 10%.

```python
import numpy as np
from sklearn.tree import DecisionTreeRegressor
from sklearn.ensemble import GradientBoostingRegressor
from sklearn.datasets import make_regression
from sklearn.model_selection import train_test_split

X, y = make_regression(n_samples=500, n_features=20, noise=15, random_state=0)
X_tr, X_te, y_tr, y_te = train_test_split(X, y, test_size=0.2, random_state=0)

f = np.full(len(y_tr), y_tr.mean())
trees = []

def rmse(a, b): return np.sqrt(np.mean((a-b)**2))

for m in range(200):
    r = y_tr - f
    t = DecisionTreeRegressor(max_depth=2, random_state=0).fit(X_tr, r)
    f += 0.1 * t.predict(X_tr)
    trees.append(t)
    if (m+1) in [50, 100, 200]:
        pred = y_tr.mean() + 0.1 * sum(t.predict(X_te) for t in trees)
        sk = GradientBoostingRegressor(n_estimators=m+1, learning_rate=0.1, max_depth=2).fit(X_tr, y_tr)
        print(f"m={m+1:3}: scratch={rmse(pred,y_te):.2f}, sklearn={rmse(sk.predict(X_te),y_te):.2f}")
```
