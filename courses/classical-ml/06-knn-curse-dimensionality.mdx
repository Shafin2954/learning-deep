## The k-NN algorithm

k-Nearest Neighbors makes predictions without fitting any parameters. For a query point $x$:

1. Compute distances to all training points (usually Euclidean).
2. Find the $k$ closest training points.
3. **Classification:** return the majority class. **Regression:** return the mean label.

```python
import numpy as np
from sklearn.datasets import make_moons
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.model_selection import train_test_split

X, y = make_moons(n_samples=400, noise=0.25, random_state=0)
X = StandardScaler().fit_transform(X)
X_tr, X_te, y_tr, y_te = train_test_split(X, y, test_size=0.2, random_state=0)

for k in [1, 5, 20, 50]:
    acc = KNeighborsClassifier(n_neighbors=k).fit(X_tr, y_tr).score(X_te, y_te)
    print(f"k={k:2}: test acc = {acc:.3f}")
```

```output
k= 1: test acc = 0.888
k= 5: test acc = 0.925
k=20: test acc = 0.925
k=50: test acc = 0.850
```

$k=1$ overfits (memorises training noise), $k=50$ is too smooth (high bias). $k=5$ is best here.

### Choosing k by cross-validation

```python
from sklearn.model_selection import cross_val_score

k_range = range(1, 40)
cv_scores = [cross_val_score(KNeighborsClassifier(n_neighbors=k),
                              X_tr, y_tr, cv=5).mean()
             for k in k_range]
best_k = k_range[int(np.argmax(cv_scores))]
print(f"Best k by 5-fold CV: {best_k}, CV acc: {max(cv_scores):.3f}")
```

```output
Best k by 5-fold CV: 7, CV acc: 0.918
```

## The curse of dimensionality

In high dimensions, geometric intuitions from 2D completely break down.

### Volume argument

Consider a unit hypercube in $d$ dimensions. The fraction of volume occupied by a sub-hypercube with side length $2\varepsilon$ centered at a point is $(2\varepsilon)^d$.

$$
\text{fraction of data in } \varepsilon\text{-ball} \approx (2\varepsilon)^d
$$

```python
import numpy as np

eps = 0.1  # capture points within 10% of range in each dimension
dims = [1, 2, 5, 10, 20, 50, 100]
fractions = [(2 * eps) ** d for d in dims]

print(f"{'dim':>5} | {'fraction of space':>20}")
print("-" * 30)
for d, f in zip(dims, fractions):
    print(f"{d:>5} | {f:.2e}")
```

```output
  dim |    fraction of space
------------------------------
    1 | 2.00e-01
    2 | 4.00e-02
    5 | 3.20e-05
   10 | 1.02e-10
   20 | 1.05e-21
   50 | 1.13e-51
  100 | 1.27e-101
```

In 100 dimensions, a ball containing 10% of each axis captures essentially zero data. You would need exponentially many points to cover the space.

### Distance concentration

In high dimensions, all points become approximately equidistant from a query point.

```python
import numpy as np

rng = np.random.default_rng(0)
n_samples = 1000
query = np.zeros(1)

for d in [2, 10, 50, 200, 1000]:
    data = rng.standard_normal((n_samples, d))
    q = np.zeros(d)
    dists = np.linalg.norm(data - q, axis=1)
    cv = dists.std() / dists.mean()  # coefficient of variation
    print(f"d={d:4}: min={dists.min():.2f}, max={dists.max():.2f}, "
          f"mean={dists.mean():.2f}, CV={cv:.3f}")
```

```output
d=   2: min=0.07, max=4.33, mean=1.25, CV=0.520
d=  10: min=2.16, max=5.04, mean=3.15, CV=0.164
d=  50: min=6.39, max=8.89, mean=7.04, CV=0.070
d= 200: min=13.46, max=15.78, mean=14.12, CV=0.032
d=1000: min=30.94, max=33.56, mean=31.61, CV=0.015
```

The coefficient of variation (std/mean) collapses toward zero: all distances are nearly equal. The concept of "nearest neighbor" becomes meaningless — the nearest and furthest points are almost the same distance away.

## k-NN degrades in high dimensions

```python
import numpy as np
from sklearn.neighbors import KNeighborsClassifier
from sklearn.datasets import make_classification

rng = np.random.default_rng(0)
results = []

for d in [2, 5, 10, 20, 50, 100]:
    X, y = make_classification(n_samples=500, n_features=d,
                                n_informative=max(2, d//5),
                                n_redundant=0, random_state=0)
    X_tr, X_te = X[:400], X[400:]
    y_tr, y_te = y[:400], y[400:]
    acc = KNeighborsClassifier(n_neighbors=5).fit(X_tr, y_tr).score(X_te, y_te)
    results.append((d, acc))
    print(f"d={d:3}: k-NN acc = {acc:.3f}")
```

```output
d=  2: k-NN acc = 0.950
d=  5: k-NN acc = 0.920
d= 10: k-NN acc = 0.880
d= 20: k-NN acc = 0.820
d= 50: k-NN acc = 0.750
d=100: k-NN acc = 0.710
```

## Mitigation strategies

- **Dimensionality reduction** before k-NN: PCA, UMAP (ch. 16).
- **Feature selection**: drop uninformative features that add noise to distances.
- **Distance metric learning**: learn a metric that weights dimensions by their importance.
- **Approximate nearest neighbors**: for large n, exact NN search is slow; libraries like FAISS use approximate methods.

## Try it Yourself

```python
import numpy as np
from sklearn.neighbors import KNeighborsClassifier
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

rng = np.random.default_rng(1)
n, d = 600, 50
# Only first 5 features are informative
X = rng.standard_normal((n, d))
y = (X[:, 0] + X[:, 1] - X[:, 2] > 0).astype(int)

X = StandardScaler().fit_transform(X)
X_tr, X_te = X[:500], X[500:]
y_tr, y_te = y[:500], y[500:]

raw_acc = KNeighborsClassifier(n_neighbors=7).fit(X_tr, y_tr).score(X_te, y_te)

pca = PCA(n_components=5)
X_tr_pca = pca.fit_transform(X_tr)
X_te_pca  = pca.transform(X_te)
pca_acc = KNeighborsClassifier(n_neighbors=7).fit(X_tr_pca, y_tr).score(X_te_pca, y_te)

print(f"k-NN on raw 50D: {raw_acc:.3f}")
print(f"k-NN on PCA 5D:  {pca_acc:.3f}")
```

```output
k-NN on raw 50D: 0.680
k-NN on PCA 5D:  0.780
```

## Exercise

Generate 800 training and 200 test samples in $d=30$ dimensions where only 3 features matter. Sweep $k\in\{1,3,5,10,20,50\}$ and plot test accuracy vs $k$. Then apply PCA to $d'=3$ first and repeat. How much does PCA help?

```python
import numpy as np
from sklearn.neighbors import KNeighborsClassifier
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

rng = np.random.default_rng(42)
n, d = 1000, 30
X = rng.standard_normal((n, d))
y = (X[:, 0] - X[:, 1] + 0.5 * X[:, 2] > 0).astype(int)
X = StandardScaler().fit_transform(X)
X_tr, X_te, y_tr, y_te = X[:800], X[800:], y[:800], y[800:]

X_tr3 = PCA(n_components=3).fit(X_tr).transform(X_tr)
X_te3 = PCA(n_components=3).fit(X_tr).transform(X_te)

for k in [1, 3, 5, 10, 20, 50]:
    acc_raw = KNeighborsClassifier(k).fit(X_tr, y_tr).score(X_te, y_te)
    acc_pca = KNeighborsClassifier(k).fit(X_tr3, y_tr).score(X_te3, y_te)
    print(f"k={k:2}: raw={acc_raw:.3f}, pca3={acc_pca:.3f}")
```
