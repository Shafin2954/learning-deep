## Generative classifiers

LDA and QDA are **generative models**: they model the joint distribution $P(x, y)$ by fitting a Gaussian for each class and applying Bayes' theorem. They differ only in whether classes share a covariance matrix.

$$
P(Y=k \mid x) \propto P(x \mid Y=k)\,P(Y=k), \quad P(x\mid Y=k) = \mathcal{N}(x;\,\mu_k,\Sigma_k)
$$

## Linear Discriminant Analysis (LDA)

LDA assumes all classes share the same covariance: $\Sigma_k = \Sigma$ for all $k$.

### Decision boundary

Taking the log posterior and comparing two classes $k$ and $l$:

$$
\log\frac{P(Y=k\mid x)}{P(Y=l\mid x)} = (\mu_k - \mu_l)^T\Sigma^{-1}x - \frac{1}{2}(\mu_k^T\Sigma^{-1}\mu_k - \mu_l^T\Sigma^{-1}\mu_l) + \log\frac{\pi_k}{\pi_l}
$$

This is **linear** in $x$ — the boundary between two classes is a hyperplane.

### Fitting LDA

1. Estimate class means: $\hat\mu_k = \frac{1}{n_k}\sum_{y_i=k}x_i$
2. Estimate pooled within-class covariance: $\hat\Sigma = \frac{1}{n-K}\sum_k\sum_{y_i=k}(x_i-\hat\mu_k)(x_i-\hat\mu_k)^T$
3. Class priors: $\hat\pi_k = n_k/n$

```python
import numpy as np
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis, QuadraticDiscriminantAnalysis
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
import matplotlib.pyplot as plt

rng = np.random.default_rng(0)

# Two classes with similar covariance (good for LDA)
X_lda = np.vstack([
    rng.multivariate_normal([0, 0], [[2, 1],[1, 2]], 150),
    rng.multivariate_normal([4, 3], [[2, 1],[1, 2]], 150),
])
y_lda = np.array([0]*150 + [1]*150)

X_tr, X_te, y_tr, y_te = train_test_split(X_lda, y_lda, test_size=0.2, random_state=0)

lda = LinearDiscriminantAnalysis()
lda.fit(X_tr, y_tr)
print(f"LDA accuracy (equal cov): {lda.score(X_te, y_te):.3f}")
print(f"Learned means:\n  class 0: {lda.means_[0].round(3)}\n  class 1: {lda.means_[1].round(3)}")
```

```output
LDA accuracy (equal cov): 0.983
Learned means:
  class 0: [-0.032  0.039]
  class 1: [ 4.017  2.983]
```

## Fisher's discriminant direction

LDA can also be viewed as finding the projection direction $w$ that maximises:

$$
J(w) = \frac{w^T S_B w}{w^T S_W w}
$$

where $S_B$ is the **between-class scatter** matrix and $S_W$ is the **within-class scatter** matrix. The optimal $w = S_W^{-1}(\mu_1 - \mu_2)$ is the Fisher discriminant direction. This makes LDA useful for **dimensionality reduction**: project $p$-dimensional data onto the $K-1$ Fisher directions.

```python
# LDA as a dimensionality reducer (3 classes → 2 Fisher directions)
from sklearn.datasets import load_iris

X_iris, y_iris = load_iris(return_X_y=True)
lda_dr = LinearDiscriminantAnalysis(n_components=2)
X_proj = lda_dr.fit_transform(X_iris, y_iris)

print(f"Original shape: {X_iris.shape}")
print(f"Projected shape: {X_proj.shape}")
print(f"Explained variance ratio: {lda_dr.explained_variance_ratio_.round(3)}")
```

```output
Original shape: (150, 4)
Projected shape: (150, 2)
Explained variance ratio: [0.992 0.008]
```

## Quadratic Discriminant Analysis (QDA)

QDA allows each class its own covariance $\Sigma_k$. The log-posterior includes a quadratic term:

$$
\log P(Y=k\mid x) \propto -\frac{1}{2}\log|\Sigma_k| - \frac{1}{2}(x-\mu_k)^T\Sigma_k^{-1}(x-\mu_k) + \log\pi_k
$$

This is **quadratic** in $x$ — the boundary is a quadratic surface (ellipse, hyperbola, parabola).

```python
import numpy as np
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis, QuadraticDiscriminantAnalysis
from sklearn.model_selection import train_test_split

rng = np.random.default_rng(1)

# Two classes with DIFFERENT covariances (QDA should win)
X_qda = np.vstack([
    rng.multivariate_normal([0, 0], [[3, 0],[0, 0.5]], 200),   # elongated along x
    rng.multivariate_normal([2, 2], [[0.5, 0],[0, 3]], 200),   # elongated along y
])
y_qda = np.array([0]*200 + [1]*200)
X_tr, X_te, y_tr, y_te = train_test_split(X_qda, y_qda, test_size=0.2, random_state=0)

for name, clf in [("LDA", LinearDiscriminantAnalysis()),
                   ("QDA", QuadraticDiscriminantAnalysis())]:
    acc = clf.fit(X_tr, y_tr).score(X_te, y_te)
    print(f"{name}: {acc:.3f}")
```

```output
LDA: 0.869
QDA: 0.934
```

QDA captures the different shapes; LDA uses a linear boundary that can't separate them as well.

## When to use each

| Situation | Prefer |
|---|---|
| Few training samples per class | LDA (shared Σ → fewer parameters) |
| Large training set, class shapes differ | QDA |
| You need feature reduction to $K-1$ dims | LDA |
| Equal class covariances | Either (LDA is simpler) |

### Rule of thumb: compare $n/p$ ratio

LDA has $O(p^2)$ parameters (one shared $\Sigma$), QDA has $O(Kp^2)$ parameters. When $n/p$ is small, LDA generalises better.

## Try it Yourself

```python
import numpy as np
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis, QuadraticDiscriminantAnalysis
from sklearn.model_selection import cross_val_score

rng = np.random.default_rng(42)
# 4-class problem in 6D; classes have different shapes
means = [[0,0,0,0,0,0], [3,0,0,0,0,0], [0,3,0,0,0,0], [3,3,0,0,0,0]]
covs  = [
    np.diag([2,0.5,1,1,1,1]),
    np.diag([0.5,2,1,1,1,1]),
    np.diag([1,1,2,0.5,1,1]),
    np.diag([1,1,0.5,2,1,1]),
]
X = np.vstack([rng.multivariate_normal(m, c, 100) for m, c in zip(means, covs)])
y = np.repeat([0,1,2,3], 100)

lda_cv = cross_val_score(LinearDiscriminantAnalysis(), X, y, cv=5).mean()
qda_cv = cross_val_score(QuadraticDiscriminantAnalysis(), X, y, cv=5).mean()
print(f"LDA 5-fold CV: {lda_cv:.3f}")
print(f"QDA 5-fold CV: {qda_cv:.3f}")
```

```output
LDA 5-fold CV: 0.835
QDA 5-fold CV: 0.913
```
