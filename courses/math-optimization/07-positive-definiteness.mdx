## Quadratic forms

A **quadratic form** maps a vector to a scalar via a symmetric matrix $A \in \mathbb{R}^{d \times d}$:

$$
q(x) = x^T A x = \sum_{i,j} A_{ij} x_i x_j
$$

The shape of this bowl (or saddle, or flat valley) is determined entirely by the eigenvalues of $A$.

| Eigenvalues | Shape | Name |
|---|---|---|
| All $> 0$ | Strict bowl (unique minimum) | **Positive definite (PD)** |
| All $\ge 0$ | Flat directions possible | **Positive semi-definite (PSD)** |
| Mixed signs | Saddle | Indefinite |
| All $\le 0$ | Inverted bowl | Negative semi-definite |

```python
import numpy as np

# PD matrix: all eigenvalues > 0
A_pd = np.array([[4.0, 1.0], [1.0, 3.0]])
eigvals = np.linalg.eigvalsh(A_pd)
print("PD matrix eigenvalues:", np.round(eigvals, 4))

# Evaluate q(x) at several points
for x in [np.array([1.0, 0.0]), np.array([0.0, 1.0]), np.array([1.0, -1.0])]:
    print(f"  q({x}) = {x @ A_pd @ x:.4f}")
```

```output
PD matrix eigenvalues: [2.6972 4.3028]
  q([1. 0.]) = 4.0000
  q([0. 1.]) = 3.0000
  q([ 1. -1.]) = 5.0000
```

All values are positive — as guaranteed for a PD matrix.

## Equivalent characterisations of PD

Three equivalent conditions for $A$ to be positive definite:

1. **All eigenvalues positive:** $\lambda_i > 0$ for all $i$.
2. **Sylvester's criterion:** all leading principal minors are positive ($\det(A_{1:k,1:k}) > 0$ for $k = 1, \ldots, d$).
3. **Cholesky factorisation exists:** $A = LL^T$ with $L$ lower-triangular and positive diagonal.

```python
import numpy as np

A = np.array([[4.0, 2.0, 1.0],
              [2.0, 5.0, 3.0],
              [1.0, 3.0, 6.0]])

# Test 1: eigenvalues
eigvals = np.linalg.eigvalsh(A)
print("Eigenvalues:", np.round(eigvals, 4))
print("All positive:", (eigvals > 0).all())

# Test 2: leading minors
minors = [np.linalg.det(A[:k, :k]) for k in range(1, 4)]
print("Leading minors:", np.round(minors, 4))
print("All positive  :", all(m > 0 for m in minors))

# Test 3: Cholesky
try:
    L = np.linalg.cholesky(A)
    print("Cholesky succeeded — A is PD")
    print("L diagonal:", np.round(np.diag(L), 4))
except np.linalg.LinAlgError:
    print("Cholesky failed — A is not PD")
```

```output
Eigenvalues: [1.2533 3.4142 10.3325]
All positive: True
Leading minors: [ 4.     16.     57.0001]
All positive  : True
Cholesky succeeded — A is PD
L diagonal: [2.     1.    2.2804]
```

## Why covariance matrices are PSD

A sample covariance matrix $C = X_c^T X_c / (n-1)$ is always PSD:

$$
x^T C x = x^T \frac{X_c^T X_c}{n-1} x = \frac{\|X_c x\|^2}{n-1} \ge 0
$$

It is PD only when $X_c$ has full column rank (i.e., $n > d$ and no perfectly collinear features). When $n < d$, at least $d - n$ eigenvalues are zero — the matrix is PSD but not PD.

```python
import numpy as np

rng = np.random.default_rng(3)

# Under-determined: n < d → PSD not PD
n, d = 50, 100
X = rng.standard_normal((n, d))
C = (X - X.mean(axis=0)).T @ (X - X.mean(axis=0)) / (n - 1)
eigvals = np.linalg.eigvalsh(C)
zero_eigvals = (eigvals < 1e-10).sum()
print(f"n={n}, d={d}: {zero_eigvals} eigenvalues are effectively 0 — matrix is PSD only")

# Over-determined: n >> d → PD
n2, d2 = 500, 10
X2 = rng.standard_normal((n2, d2))
C2 = (X2 - X2.mean(axis=0)).T @ (X2 - X2.mean(axis=0)) / (n2 - 1)
eigvals2 = np.linalg.eigvalsh(C2)
print(f"n={n2}, d={d2}: min eigenvalue = {eigvals2.min():.4f} — matrix is PD")
```

```output
n=50, d=100: 51 eigenvalues are effectively 0 — matrix is PSD only
n=500, d=10: min eigenvalue = 0.7823 — matrix is PD
```

## Cholesky factorisation

For a PD matrix $A$, the **Cholesky factorisation** $A = LL^T$ (with $L$ lower-triangular, positive diagonal) exists and is unique. It is twice as fast as LU decomposition and numerically the most stable path for solving $Ax = b$ when $A$ is symmetric and PD.

```python
import numpy as np

rng = np.random.default_rng(0)
d = 5

# Build a guaranteed-PD matrix: A = Q^T Q + εI
Q = rng.standard_normal((d, d))
A = Q.T @ Q + 0.5 * np.eye(d)

L = np.linalg.cholesky(A)
print("L (lower triangular):")
print(np.round(L, 4))
print("\nReconstruction error:", np.max(np.abs(L @ L.T - A)))

# Solve Ax = b via Cholesky: L L^T x = b
b = rng.standard_normal(d)
y = np.linalg.solve(L, b)           # forward substitution: Ly = b
x = np.linalg.solve(L.T, y)         # back substitution: L^T x = y

residual = np.linalg.norm(A @ x - b)
print(f"\nResidual ||Ax - b|| = {residual:.2e}")
```

### Adding regularisation to make a PSD matrix PD

Ridge regularisation $A + \varepsilon I$ shifts all eigenvalues up by $\varepsilon$, guaranteeing PD:

```python
import numpy as np

rng = np.random.default_rng(7)
n, d = 30, 50
X = rng.standard_normal((n, d))
C = X.T @ X / (n - 1)   # PSD: rank at most 30

eps = 1e-4
C_reg = C + eps * np.eye(d)

try:
    np.linalg.cholesky(C_reg)
    print(f"With epsilon={eps}: Cholesky succeeded (C_reg is PD)")
except np.linalg.LinAlgError:
    print("Still not PD — increase epsilon")
```

```output
With epsilon=0.0001: Cholesky succeeded (C_reg is PD)
```

```python
import numpy as np

rng = np.random.default_rng(42)
d = 4
Q = rng.standard_normal((d, d))
eps = 0.1

A = Q.T @ Q + eps * np.eye(d)
eigvals = np.linalg.eigvalsh(A)
print("Eigenvalues:", np.round(eigvals, 4))

print("Cholesky diagonal:", np.round(np.diag(L), 4))
print("Reconstruction max error:", np.max(np.abs(L @ L.T - A)))
```

## Exercise

Estimate a covariance matrix from 20 samples of a 50-dimensional Gaussian. Determine whether it is PD or only PSD. Add the smallest regularisation $\varepsilon$ (search over powers of 10) that makes it PD, and verify with Cholesky.

<details>
<summary>Solution</summary>

```python
import numpy as np

rng = np.random.default_rng(0)
X = rng.standard_normal((20, 50))
C = (X - X.mean(axis=0)).T @ (X - X.mean(axis=0)) / 19

eigvals = np.linalg.eigvalsh(C)
print(f"Min eigenvalue: {eigvals.min():.6f} — {'PD' if eigvals.min() > 0 else 'PSD only'}")

for exp in range(-8, 1):
    eps = 10.0 ** exp
    try:
        np.linalg.cholesky(C + eps * np.eye(50))
        print(f"epsilon = 1e{exp}: Cholesky succeeded → PD")
        break
    except np.linalg.LinAlgError:
        print(f"epsilon = 1e{exp}: still not PD")
```

With n=20 and d=50, you'll find 31 zero eigenvalues. A regularisation of $10^{-10}$ or smaller typically suffices.
</details>
