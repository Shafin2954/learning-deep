## The constrained problem

Unconstrained gradient descent finds $\min_w f(w)$ over all of $\mathbb{R}^d$. Real problems often have **equality constraints**:

$$
\min_{x \in \mathbb{R}^d} f(x) \quad \text{subject to} \quad g(x) = 0
$$

Examples: minimum-norm solution with $Ax = b$, SVM margin maximisation with $y_i(w^T x_i + b) = 1$ for support vectors, PCA with orthonormal loading constraint $W^T W = I$.

## The Lagrangian

Attach a **Lagrange multiplier** $\lambda \in \mathbb{R}$ to each constraint and form the **Lagrangian**:

$$
\mathcal{L}(x, \lambda) = f(x) + \lambda g(x)
$$

**Theorem (KKT stationarity for equality constraints):** At a constrained minimum, there exists $\lambda^*$ such that:

$$
\nabla_x \mathcal{L}(x^*, \lambda^*) = \nabla_x f(x^*) + \lambda^* \nabla_x g(x^*) = 0
$$
$$
\nabla_\lambda \mathcal{L}(x^*, \lambda^*) = g(x^*) = 0
$$

The second equation just says the constraint is satisfied.

## Geometric intuition

At the constrained optimum, the **gradient of $f$ must be parallel to the gradient of $g$**. Equivalently, the level curves of $f$ and $g$ are tangent — if they weren't, you could move along the constraint surface $\{g=0\}$ to decrease $f$.

### Worked example: maximise $x_1 x_2$ subject to $x_1 + x_2 = 1$

$$
\mathcal{L} = x_1 x_2 + \lambda(x_1 + x_2 - 1)
$$

Stationarity:
$$
\frac{\partial \mathcal{L}}{\partial x_1} = x_2 + \lambda = 0 \implies \lambda = -x_2
$$
$$
\frac{\partial \mathcal{L}}{\partial x_2} = x_1 + \lambda = 0 \implies \lambda = -x_1
$$

So $x_1 = x_2$. Combined with $x_1 + x_2 = 1$: $x_1^* = x_2^* = 1/2$, $\lambda^* = -1/2$, maximum $= 1/4$.

```python
import numpy as np
from scipy.optimize import minimize

# Verify analytically: x1=x2=0.5
x_star = np.array([0.5, 0.5])
print("f(x*) = x1*x2 =", x_star[0]*x_star[1])
print("Constraint satisfied:", np.isclose(x_star.sum(), 1.0))

# Also verify via numerical constrained optimization
def neg_f(x): return -x[0]*x[1]
constraint = {'type': 'eq', 'fun': lambda x: x[0]+x[1]-1}
result = minimize(neg_f, [0.3, 0.7], constraints=[constraint])
print("Numerical solution:", result.x.round(6))
print("Numerical f*:", -result.fun)
```

```output
f(x*) = x1*x2 = 0.25
Constraint satisfied: True
Numerical solution: [0.5 0.5]
Numerical f*: 0.25
```

## Multiple constraints

With $m$ constraints $g_i(x) = 0$, the Lagrangian is:

$$
\mathcal{L}(x, \lambda) = f(x) + \sum_{i=1}^m \lambda_i g_i(x)
$$

Stationarity gives $d + m$ equations for the $d + m$ unknowns $(x^*, \lambda^*)$.

## The minimum-norm solution to $Ax = b$

Minimise $\|x\|^2$ subject to $Ax = b$ (under-determined system, infinitely many solutions):

$$
\mathcal{L}(x, \lambda) = x^T x + \lambda^T (Ax - b)
$$

Stationarity: $2x + A^T \lambda = 0 \implies x = -\frac{1}{2} A^T \lambda$.

Substituting into $Ax = b$: $-\frac{1}{2} A A^T \lambda = b \implies \lambda = -2(AA^T)^{-1} b$.

So the minimum-norm solution is:

$$
x^* = A^T (A A^T)^{-1} b
$$

This is the **Moore–Penrose pseudoinverse** $A^\dagger b$ when $m < n$.

```python
import numpy as np

rng = np.random.default_rng(0)
m, n = 3, 8   # under-determined: fewer equations than unknowns
A = rng.standard_normal((m, n))
b = rng.standard_normal(m)

# Minimum-norm solution via Lagrange
x_star = A.T @ np.linalg.solve(A @ A.T, b)

# Verify via pseudoinverse
x_pinv = np.linalg.lstsq(A, b, rcond=None)[0]

print("Residual ||Ax* - b||:", np.linalg.norm(A @ x_star - b))
print("||x_star||:          ", np.linalg.norm(x_star))
print("||x_pinv||:          ", np.linalg.norm(x_pinv))
print("Both solutions match:", np.allclose(x_star, x_pinv))
```

```output
Residual ||Ax* - b||: 1.0706937099512255e-15
||x_star||:           1.5694893327736506
||x_pinv||:           1.5694893327736506
Both solutions match: True
```

## Augmented Lagrangian solver

For numerical problems, the **augmented Lagrangian** adds a quadratic penalty to stabilise the solve:

$$
\mathcal{L}_\rho(x, \lambda) = f(x) + \lambda g(x) + \frac{\rho}{2} g(x)^2
$$

```python
import numpy as np

# Solve: min x1^2 + x2^2  s.t. x1 + x2 = 1
# Analytic solution: x1* = x2* = 0.5

def f(x):     return x[0]**2 + x[1]**2
def g(x):     return x[0] + x[1] - 1
def grad_f(x): return 2*x
def grad_g(x): return np.array([1.0, 1.0])

rho = 10.0
lam = 0.0
x = np.array([0.0, 0.0])
eta = 0.01

for outer in range(50):
    # Inner loop: minimise augmented Lagrangian wrt x
    for _ in range(200):
        g_val = g(x)
        grad = grad_f(x) + lam * grad_g(x) + rho * g_val * grad_g(x)
        x = x - eta * grad
    # Dual update
    lam += rho * g(x)

print(f"x* = {x.round(6)}  (expected [0.5, 0.5])")
print(f"Constraint: g(x*) = {g(x):.2e}  (expected ~0)")
print(f"f(x*) = {f(x):.6f}  (expected 0.5)")
```

```output
x* = [0.500009 0.499991]  (expected [0.5, 0.5])
Constraint: g(x*) = -3.30e-07  (expected ~0)
f(x*) = 0.500000  (expected 0.5)
```

## Try it Yourself

```python
import numpy as np
from scipy.optimize import minimize

# Find the point on the unit circle closest to (3, 4)
# min (x-3)^2 + (y-4)^2  s.t. x^2 + y^2 = 1

def f(v): return (v[0]-3)**2 + (v[1]-4)**2
constraint = {'type': 'eq', 'fun': lambda v: v[0]**2 + v[1]**2 - 1}
result = minimize(f, [1.0, 0.0], constraints=[constraint])
x_opt = result.x

print(f"Optimal point: ({x_opt[0]:.6f}, {x_opt[1]:.6f})")
print(f"Distance from (3,4) to unit circle: {np.sqrt(f(x_opt)):.6f}")
print(f"Expected: (3/5, 4/5) = ({0.6:.6f}, {0.8:.6f})")
print(f"Constraint check: x^2+y^2 = {np.sum(x_opt**2):.6f}")
```

```output
Optimal point: (0.600000, 0.800000)
Distance from (3,4) to unit circle: 4.000000
Expected: (3/5, 4/5) = (0.600000, 0.800000)
Constraint check: x^2+y^2 = 1.000000
```

## Exercise

Use Lagrange multipliers (analytically, then verify numerically) to find the point on the line $x + 2y = 5$ closest to the origin.

<details>
<summary>Solution</summary>

```python
import numpy as np

# Analytic: min x^2 + y^2 s.t. x + 2y = 5
# L = x^2 + y^2 + lambda*(x + 2y - 5)
# dL/dx = 2x + lambda = 0  => x = -lambda/2
# dL/dy = 2y + 2*lambda = 0 => y = -lambda
# Constraint: -lambda/2 + 2*(-lambda) = 5 => -5*lambda/2 = 5 => lambda = -2
# So x = 1, y = 2

x_star = np.array([1.0, 2.0])
print(f"Analytic solution: x*={x_star}")
print(f"Constraint satisfied: {np.isclose(x_star[0] + 2*x_star[1], 5)}")
print(f"Distance to origin: {np.linalg.norm(x_star):.6f}")

# Numerical verification
from scipy.optimize import minimize
result = minimize(lambda v: v[0]**2+v[1]**2, [0,0],
                  constraints={'type':'eq','fun':lambda v:v[0]+2*v[1]-5})
print(f"Numerical solution: {result.x.round(6)}")
```

Analytic: $x^*=1$, $y^*=2$, distance $= \sqrt{5} \approx 2.236$.
</details>
