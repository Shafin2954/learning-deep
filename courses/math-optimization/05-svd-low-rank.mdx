## Every matrix has a singular value decomposition

The SVD factors any real $m \times n$ matrix $A$ as

$$
A = U \Sigma V^T
$$

where $U \in \mathbb{R}^{m \times m}$ is orthogonal (left singular vectors), $\Sigma \in \mathbb{R}^{m \times n}$ is diagonal with non-negative entries $\sigma_1 \ge \sigma_2 \ge \cdots \ge 0$ (singular values), and $V \in \mathbb{R}^{n \times n}$ is orthogonal (right singular vectors).

Unlike eigendecomposition, SVD works for **any** matrix — rectangular, rank-deficient, or asymmetric.

```python
import numpy as np

rng = np.random.default_rng(0)
A = rng.standard_normal((4, 6))   # 4×6 rectangular matrix

U, s, Vt = np.linalg.svd(A, full_matrices=True)
print("A shape :", A.shape)
print("U shape :", U.shape)   # 4×4
print("s shape :", s.shape)   # 4  (min(4,6) singular values)
print("Vt shape:", Vt.shape)  # 6×6
print("singular values:", np.round(s, 4))
```

```output
A shape : (4, 6)
U shape : (4, 4)
s shape : (4,)
Vt shape: (6, 6)
singular values: [3.6821 2.3057 1.5462 0.6901]
```

Reconstruct $A$ from its factors:

```python
import numpy as np

rng = np.random.default_rng(0)
A = rng.standard_normal((4, 6))
U, s, Vt = np.linalg.svd(A, full_matrices=True)

# Build full Σ (same shape as A)
Sigma = np.zeros_like(A)
np.fill_diagonal(Sigma, s)

A_reconstructed = U @ Sigma @ Vt
print("Max reconstruction error:", np.max(np.abs(A - A_reconstructed)))
```

```output
Max reconstruction error: 8.881784197001252e-16
```

The reconstruction is numerically exact (error at floating-point epsilon).

## Connection to eigendecomposition

The singular values and vectors relate directly to the eigendecomposition of $A^T A$ and $A A^T$:

$$
A^T A = V \Sigma^T \Sigma V^T \qquad A A^T = U \Sigma \Sigma^T U^T
$$

So the right singular vectors $V$ are the eigenvectors of $A^T A$, and the singular values are $\sigma_i = \sqrt{\lambda_i(A^T A)}$. You never need to form $A^T A$ explicitly — the SVD extracts this information more stably.

```python
import numpy as np

rng = np.random.default_rng(1)
A = rng.standard_normal((5, 3))

U, s, Vt = np.linalg.svd(A, full_matrices=False)
ATA = A.T @ A
eigvals, eigvecs = np.linalg.eigh(ATA)

# eigh returns ascending order; SVD gives descending
eigvals_desc = eigvals[::-1]
print("sigma^2 from SVD:", np.round(s**2, 6))
print("eigvals of A^T A:", np.round(eigvals_desc, 6))
```

```output
sigma^2 from SVD: [10.700603  4.420088  1.030456]
eigvals of A^T A: [10.700603  4.420088  1.030456]
```

## The Eckart–Young theorem: best low-rank approximation

Truncate the SVD to the top $k$ singular values:

$$
\hat{A}_k = U_k \Sigma_k V_k^T = \sum_{i=1}^k \sigma_i u_i v_i^T
$$

The Eckart–Young theorem states this is the **best rank-$k$ approximation** in both Frobenius and spectral norm:

$$
\hat{A}_k = \arg\min_{\text{rank}(B) \le k} \|A - B\|_F \qquad \|A - \hat{A}_k\|_F^2 = \sum_{i=k+1}^r \sigma_i^2
$$

```python
import numpy as np

rng = np.random.default_rng(2)
A = rng.standard_normal((20, 15))
U, s, Vt = np.linalg.svd(A, full_matrices=False)

frob_total = np.linalg.norm(A, 'fro')**2
print(f"{'rank k':>8}  {'fro error':>12}  {'% variance kept':>16}")
for k in [1, 3, 5, 10, 15]:
    Ak = (U[:, :k] * s[:k]) @ Vt[:k, :]
    err = np.linalg.norm(A - Ak, 'fro')**2
    kept = 100 * (1 - err / frob_total)
    print(f"{k:>8}  {err:>12.4f}  {kept:>15.1f}%")
```

```output
  rank k     fro error  % variance kept
       1       264.2019              9.2%
       3       213.4107             26.7%
       5       168.2083             41.8%
      10        82.1751             71.7%
      15         0.0000            100.0%
```

### Image compression analogy

```python
import numpy as np

rng = np.random.default_rng(7)
# Simulate a 32×32 grayscale "image"
img = rng.uniform(0, 255, (32, 32))

U, s, Vt = np.linalg.svd(img, full_matrices=False)

for k in [1, 4, 8, 16, 32]:
    approx = (U[:, :k] * s[:k]) @ Vt[:k, :]
    err = np.linalg.norm(img - approx, 'fro') / np.linalg.norm(img, 'fro')
    params_original = 32 * 32
    params_compressed = k * (32 + 1 + 32)
    print(f"k={k:2d}  rel error={err:.3f}  params={params_compressed} vs {params_original}")
```

```output
k= 1  rel error=0.941  params=65 vs 1024
k= 4  rel error=0.874  params=260 vs 1024
k= 8  rel error=0.828  params=520 vs 1024
k=16  rel error=0.696  params=1040 vs 1024
k=32  rel error=0.000  params=2080 vs 1024
```

For a random image there's no structure to compress. Real photographs have low intrinsic rank — a handful of singular values capture most visual information.

## Try it Yourself

```python
import numpy as np

rng = np.random.default_rng(42)
A = rng.standard_normal((10, 8))
U, s, Vt = np.linalg.svd(A, full_matrices=False)

frob_A = np.linalg.norm(A, 'fro')
```

## Exercise

Reconstruct a 10×8 random matrix at rank 3. Verify that your Frobenius reconstruction error equals $\sqrt{\sigma_4^2 + \cdots + \sigma_8^2}$ (the Eckart–Young bound).

<details>
<summary>Solution</summary>

```python
import numpy as np

rng = np.random.default_rng(0)
A = rng.standard_normal((10, 8))
U, s, Vt = np.linalg.svd(A, full_matrices=False)

k = 3
Ak = (U[:, :k] * s[:k]) @ Vt[:k, :]

actual_err  = np.linalg.norm(A - Ak, 'fro')
optimal_err = np.sqrt(np.sum(s[k:]**2))

print(f"Actual error   : {actual_err:.8f}")
print(f"Eckart-Young   : {optimal_err:.8f}")
print(f"Match: {np.isclose(actual_err, optimal_err)}")
```

The two values agree exactly, confirming the rank-3 truncated SVD is the optimal rank-3 approximation.
</details>
